---
title: "Export Database Table via phpMyAdmin in cPanel"
description: "Learn how to export a database table using phpMyAdmin within your cPanel account for backups or data transfer."
---

Exporting a database table in cPanel using phpMyAdmin is a straightforward process that allows you to create backups or transfer your data.

<Steps>
  <Step title="Log in to cPanel">
    Begin by logging into your cPanel account.
  </Step>
  <Step title="Access phpMyAdmin">
    In the **Databases** section, click on the **phpMyAdmin** icon.
    <Frame>
      <img 
        src="/images/kb/phpmyadmin-icon.gif" 
        alt="phpMyAdmin icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select Database">
    From the navigation tree in the left sidebar, select the database you wish to export.
    <Frame>
      <img 
        src="/images/kb/phpmyadmin-expand-db.gif" 
        alt="Expanding a database in phpMyAdmin"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Export">
    Click on the **Export** tab.
    <Frame>
      <img 
        src="/images/kb/phpmyadmin-export.gif" 
        alt="Export tab in phpMyAdmin"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Confirm Export">
    Ensure the **SQL** format is selected, then click the **Export** button.
  </Step>
</Steps>

A SQL file containing your database table will be downloaded to your computer. The download time will vary depending on the size of the file.
