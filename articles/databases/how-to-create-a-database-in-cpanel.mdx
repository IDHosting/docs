---
title: "Create a Database in cPanel"
description: "A guide on how to create a new database using your cPanel account."
---

Creating a database in cPanel is a quick and straightforward process. This guide will walk you through the steps to set up your database easily.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account using your provided credentials.
  </Step>
  <Step title="Navigate to Database Management">
    In the **Databases** section, click on **Manage My Databases**.

    <Frame>
      <img
        src="/images/kb/mysql-databases-icon.gif"
        alt="MySQL Databases Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Enter Database Name">
    In the **New Database** field, enter a name for the new database you wish to create.

    <Frame>
      <img
        src="/images/kb/create-db.gif"
        alt="Creating a new database in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Create the Database">
    Click on the **Create Database** button.

    <Frame>
      <img
        src="/images/kb/paper-db-added-successfully.gif"
        alt="Database added successfully confirmation"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

We hope this tutorial has been helpful in creating your database.
