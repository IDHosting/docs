---
title: "Import a Database via phpMyAdmin in cPanel"
description: "Learn how to import a database into your cPanel account using phpMyAdmin."
---

Importing a database in cPanel using phpMyAdmin is a straightforward process that allows you to restore or migrate your data with ease. Follow these steps to upload your SQL file and ensure your database is up and running.

<Steps>
  <Step title="Log in to cPanel">
    Begin by logging into your cPanel account.
  </Step>
  <Step title="Access phpMyAdmin">
    Navigate to the **Databases** section and click on the **phpMyAdmin** icon.
    <Frame>
      <img src="/images/kb/phpmyadmin-icon.gif" alt="phpMyAdmin icon in cPanel" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Select Database">
    In the left sidebar, expand the navigation tree and select the database you wish to import data into.
    <Frame>
      <img src="/images/kb/phpmyadmin-expand-db.gif" alt="Expanding a database in phpMyAdmin" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Navigate to Import Tab">
    Once the database is selected, click on the **Import** tab.
    <Frame>
      <img src="/images/kb/phpmyadmin-import.gif" alt="Import tab in phpMyAdmin" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Choose SQL File">
    Under the **File to import** section, click the **Choose File** button. Locate and select the SQL file you want to import from your computer.
    <Frame>
      <img src="/images/kb/phpmyadmin-import2.gif" alt="Choosing the SQL file to import" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Complete Import">
    After selecting your SQL file, scroll down to the bottom of the page and click the **Import** button.
    <Note>If the import is successful, you will see a success message indicating that your SQL file has been uploaded.</Note>
    <Frame>
      <img src="/images/kb/phpmyadmin-successfully.gif" alt="Successful import message in phpMyAdmin" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
</Steps>

We hope this tutorial was helpful in guiding you through the database import process.
