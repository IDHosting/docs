---
title: "Edit a File in the cPanel File Manager"
description: "Learn how to edit website files directly within the cPanel File Manager."
---

You can use cPanel's built-in file manager to edit files on your website.

<Steps>
  <Step title="Log in to cPanel">
    Access your cPanel account by logging in.
  </Step>
  <Step title="Open File Manager">
    In the **Files** section, click on **File Manager**.
    <Frame>
      <img
        src="/images/kb/filemanager-icon.gif"
        alt="cPanel File Manager Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Navigate to the File">
    Browse to the directory where your file is located.
  </Step>
  <Step title="Access Edit Options">
    Right-click on the file you wish to edit. Then, select either **Edit** or **HTML Edit** from the context menu that appears.
    <Frame>
      <img
        src="/images/kb/context-menu.gif"
        alt="File Manager Context Menu with Edit options"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Confirm Editing Mode">
    In the **HTML Editor** window, click on **Edit** to confirm you want to edit the file.
    <Frame>
      <img
        src="/images/kb/filemanager-edit-menu.gif"
        alt="HTML Editor confirmation dialog"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Make and Save Changes">
    Modify the content of your file as needed. Once you are finished, click the **Save** button.
    <Frame>
      <img
        src="/images/kb/edit-file-in-filemanager.gif"
        alt="Editing a file in cPanel File Manager"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

Your file has now been successfully edited.
