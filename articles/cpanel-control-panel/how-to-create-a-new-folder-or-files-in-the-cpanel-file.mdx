---
title: "Create a New Folder or Files in the cPanel File Manager"
description: "Learn how to create new folders and files directly within your cPanel File Manager."
---

The cPanel File Manager is a convenient tool for managing your website's files. This guide will walk you through the steps to create new folders and files.

## Creating a New Folder

<Steps>
  <Step title="Log in to cPanel">
    Access your cPanel account by logging in with your credentials.
  </Step>
  <Step title="Open File Manager">
    In the **Files** section of your cPanel dashboard, click on the **File Manager** icon.
    <Frame>
      <img 
        src="/images/kb/filemanager-icon.gif" 
        alt="cPanel File Manager Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Navigate to Directory">
    Browse to the specific directory where you wish to create your new folder.
  </Step>
  <Step title="Initiate Folder Creation">
    Click the **+Folder** button.
    <Frame>
      <img 
        src="/images/kb/filemanager-newfolder.gif" 
        alt="cPanel File Manager +Folder Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Name and Create Folder">
    In the **New Folder Name** field, enter the desired name for your folder (e.g., `my-folder`). Then, click on **Create New Folder**.
    <Frame>
      <img 
        src="/images/kb/filemanager-create-folder.gif" 
        alt="cPanel File Manager Create New Folder Dialog"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

## Creating a New File

<Steps>
  <Step title="Navigate to Directory">
    Ensure you are in the directory where you want to create the new file.
  </Step>
  <Step title="Initiate File Creation">
    Click on the **+File** icon.
    <Frame>
      <img 
        src="/images/kb/filemanager-newfiletxt.gif" 
        alt="cPanel File Manager +File Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Name and Create File">
    In the **New File Name** field, enter the desired name for your file (e.g., `MyFile.txt`). Then, click on **Create New File**.
  </Step>
</Steps>

That's it! You have successfully created a new folder and a new file in your cPanel File Manager.
