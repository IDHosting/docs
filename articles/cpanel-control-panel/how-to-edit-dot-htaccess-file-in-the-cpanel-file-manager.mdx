---
title: "Edit .htaccess File in cPanel File Manager"
description: "A guide on how to edit your .htaccess file using the cPanel File Manager."
---

This article will guide you through the process of editing your `.htaccess` file using the cPanel File Manager.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account through your hosting provider's portal.
  </Step>
  <Step title="Navigate to File Manager">
    In the **Files** section, click on **File Manager**.
    <Frame>
      <img src="/images/kb/filemanager-icon.gif" alt="cPanel File Manager Icon" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Locate the .htaccess file">
    Navigate to the directory where your `.htaccess` file is located. To edit the `.htaccess` file for your root directory, go to the `public_html` directory.
  </Step>
  <Step title="Show Hidden Files (if necessary)">
    If the `.htaccess` file is not visible, click on **Settings** in the top right corner of the File Manager. In the settings window, check the box next to **Show Hidden Files (Dotfiles)** and click **Save**. The `.htaccess` file should now be visible. If it's still not there, you can create a new one.
    <Frame>
      <img src="/images/kb/hidden-htaccess-paperlantern.gif" alt="cPanel File Manager Settings to show hidden files" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Open the .htaccess file for editing">
    Right-click on the `.htaccess` file and select **Edit** from the context menu.
    <Frame>
      <img src="/images/kb/filemanager-htaccess.gif" alt="Right-click menu on .htaccess file" style={{ width: '100%', height: 'auto' }} />
    </Frame>
    In the new window that appears, click on **Edit**.
    <Frame>
      <img src="/images/kb/filemanager-htaccess-edit-menu.gif" alt="Edit button in the .htaccess file context menu" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Make changes and save">
    Make the desired changes to your `.htaccess` file and click on **Save Changes**.
    <Frame>
      <img src="/images/kb/filemanager--htaccess-edit.gif" alt=".htaccess file editor with Save Changes button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
</Steps>

<Note>
Always back up your `.htaccess` file before making any changes. Incorrect modifications can lead to website errors.
</Note>
