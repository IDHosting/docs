---
title: "Password Protect a Directory in cPanel"
description: "Learn how to secure specific directories on your website using cPanel's Directory Privacy feature."
---

You can easily protect directories on your website with a username and password using cPanel's **Directory Privacy** tool. This adds an extra layer of security to sensitive content.

## Steps to Password Protect a Directory

<Steps>
  <Step title="Log in to cPanel">
    Access your cPanel account by navigating to your cPanel login URL and entering your credentials.
  </Step>
  <Step title="Navigate to Directory Privacy">
    Once logged in, scroll down to the **Files** section and click on **Directory Privacy**.
    <Frame>
      <img 
        src="/images/kb/password-protected-directories.gif" 
        alt="cPanel Directory Privacy option"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select the Directory">
    You will see a list of directories. Click on the name of the directory you wish to protect to open it.
    <Frame>
      <img 
        src="/images/kb/paper-folder-icon-passprotect.gif" 
        alt="Selecting a directory in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Edit Directory Settings">
    Next to the directory you've selected, click the **Edit** link.
    <Frame>
      <img 
        src="/images/kb/pass-pro.gif" 
        alt="Edit button for directory privacy"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Enable Password Protection">
    Under the **Security Settings** section, check the box labeled **Password protect this directory**. Then, click **Save**.
    <Frame>
      <img 
        src="/images/kb/password-settings.gif" 
        alt="Enabling password protection for a directory"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Go Back">
    After saving, click the **Go back** link to return to the directory listing.
    <Frame>
      <img 
        src="/images/kb/goback.gif" 
        alt="Go back button after saving directory privacy settings"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Create a User">
    Scroll down to the **Create User** section. Enter a **Username** and **Password** for accessing the protected directory. You can also use the **Password Generator** to create a strong, random password. Make sure to copy and save this password securely (e.g., in Notepad) if you use the generator. Click **Save** once you have entered the credentials.
    <Frame>
      <img 
        src="/images/kb/password-pro-user-gen.gif" 
        alt="Creating a user for password protected directory"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

## Accessing the Protected Directory

Once these steps are completed, any attempt to access the protected directory via its URL (e.g., `www.example.com/myfolder`) will prompt the user for the username and password you just created.

<Frame>
  <img 
    src="/images/kb/prompted-for-login.gif" 
    alt="Browser prompt for username and password"
    style={{ width: '100%', height: 'auto' }}
  />
</Frame>

By following these instructions, you have successfully password-protected a directory in cPanel, adding an essential layer of security to your website's content.
