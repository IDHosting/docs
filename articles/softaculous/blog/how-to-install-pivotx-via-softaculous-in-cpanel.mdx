---
title: "Install PivotX via Softaculous in cPanel"
description: "A guide on how to easily install PivotX using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to install PivotX on your hosting account.

<Steps>
  <Step title="Log in to cPanel">
    Begin by logging into your cPanel account.
  </Step>
  <Step title="Access Softaculous Apps Installer">
    Navigate to the **Software** section and click on **Softaculous Apps Installer**.
    <Frame>
      <img
        src="/images/kb/softaculous-icon.gif"
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Search for PivotX">
    Once in the Softaculous interface, use the search box on the left-hand side to type "PivotX". Click on the PivotX link when it appears.
    <Frame>
      <img
        src="/images/kb/pivotx-softaculous.gif"
        alt="Searching for PivotX in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the PivotX installation process.
    <Frame>
      <img
        src="/images/kb/pivotx-install-button.gif"
        alt="PivotX Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the required fields under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the appropriate protocol and domain. If you wish to install PivotX in a subdirectory (e.g., `example.com/directory`), enter the directory name in the "In Directory" field. Leave it blank to install in your main domain (e.g., `example.com`).
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Softaculous general installation screen for URL setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide the following information:
    <ul>
      <li>
        <strong>Site Name:</strong> Enter the name for your website (e.g., "My First Website").
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    In the **Admin Account** section, enter your administrative details:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a username for your PivotX administrator. For security, avoid common usernames like "admin" or "administrator".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and secure password.
      </li>
      <li>
        <strong>Nickname:</strong> Enter a nickname for the admin user.
      </li>
      <li>
        <strong>Admin Email:</strong> Provide your email address.
      </li>
    </ul>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as Notepad. You will need these credentials to log in to your PivotX admin area later.
    </Note>
    <Frame>
      <img
        src="/images/kb/softaculous-install-screen-nickname.gif"
        alt="Softaculous installation screen for admin account details"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Language">
    Select the desired language for your PivotX blog. The default language is English.
    <Frame>
      <img
        src="/images/kb/softaculous-choose-language.gif"
        alt="Softaculous screen for choosing the language"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    You can configure **Advanced Options** if you wish to change the default database name or set up automated backups. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation.
    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous final install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
