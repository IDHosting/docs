---
title: "Install HTMLy via Softaculous in cPanel"
description: "A guide on how to install HTMLy using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to easily install HTMLy on your cPanel account.

<Steps>
  <Step title="Log in to your cPanel account">
    Begin by accessing your cPanel dashboard.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    Once logged in, locate the **Software** section and click on **Softaculous Apps Installer**.

    <Frame>
      <img
        src="/images/kb/softaculous-icon.gif"
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for HTMLy">
    On the left-hand sidebar of the Softaculous interface, use the search box to type **HTMLy**. Click on the **HTMLy** link that appears in the search results.

    <Frame>
      <img
        src="/images/kb/htmly-softaculous.gif"
        alt="Searching for HTMLy in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the HTMLy installation process.

    <Frame>
      <img
        src="/images/kb/htmly-install-button.gif"
        alt="HTMLy Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    In the **Software Setup** section, configure the following:
    *   **Choose Installation URL:** Select the appropriate protocol (e.g., `http://` or `https://`) and your domain. If you wish to install HTMLy in a subdirectory (e.g., `example.com/blog`), enter the directory name in the **In Directory** field. Leave this field blank to install HTMLy on your main domain (e.g., `example.com`).

        <Frame>
          <img
            src="/images/kb/softaculous-general-install-screen.gif"
            alt="HTMLy Software Setup in Softaculous"
            style={{ width: '100%', height: 'auto' }}
          />
        </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide your website's name in the **Site Name** field (e.g., `My First Website`).
  </Step>
  <Step title="Set Up Admin Account">
    In the **Admin Account** section, enter your administrator credentials:
    *   **Admin Username:** Choose a username for your HTMLy admin panel. For security, avoid using common usernames like "admin" or "administrator".
    *   **Admin Password:** Create a strong and secure password.

        <Frame>
          <img
            src="/images/kb/softaculous-screen-two-install.gif"
            alt="HTMLy Admin Account setup in Softaculous"
            style={{ width: '100%', height: 'auto' }}
          />
        </Frame>

    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as Notepad. You will need these credentials to log in to your HTMLy admin area later.
    </Note>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database details or set up automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the HTMLy installation.

    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Final Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
