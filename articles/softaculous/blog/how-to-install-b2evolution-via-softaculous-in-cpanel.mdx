---
title: "Install b2evolution via Softaculous in cPanel"
description: "A guide on how to easily install b2evolution using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to install b2evolution on your hosting account.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel dashboard.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel, click on **Softaculous Apps Installer**.

    <Frame>
      <img
        src="/images/kb/softaculous-icon.gif"
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for b2evolution">
    On the left-hand side of the Softaculous interface, type `b2evolution` into the search box. Click on the `b2evolution` link that appears.

    <Frame>
      <img
        src="/images/kb/b2evolution-softaculous.gif"
        alt="Searching for b2evolution in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.

    <Frame>
      <img
        src="/images/kb/b2evolution-install-button.gif"
        alt="b2evolution Install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    *   **Choose Installation URL**: Select the desired protocol (e.g., `http` or `https`) and your domain.
        *   **In Directory**: If you wish to install b2evolution in a subdirectory (e.g., `example.com/blog`), enter the directory name here. Leave it blank to install in the root of your domain (e.g., `example.com`).

    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Softaculous general installation screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, provide the following details:
    *   **Admin Username**: Choose a username for your b2evolution administrator account. For security, avoid common usernames like "admin" or "administrator".
    *   **Admin Password**: Create a strong and secure password.
    *   **Admin Email**: Enter your email address.

    <Frame>
      <img
        src="/images/kb/softaculous-general-admin-fields.gif"
        alt="Softaculous admin account fields"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>

    <Note>
      It is highly recommended to copy your admin username and password to a secure location (like Notepad) immediately after setting them. You will need these credentials to log in to your b2evolution blog's administration area later.
    </Note>
  </Step>
  <Step title="Choose Language">
    Select the preferred language for your b2evolution blog from the **Choose Language** dropdown. The default language is English.

    <Frame>
      <img
        src="/images/kb/softaculous-choose-language.gif"
        alt="Softaculous language selection"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows for further customization, such as changing the database name or configuring automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to begin the installation process.

    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous final install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
