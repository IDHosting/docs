---
title: "Install Dotclear via Softaculous in cPanel"
description: "A guide on how to install Dotclear using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to install Dotclear on your cPanel account.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account through your hosting provider's portal.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel dashboard, click on **Softaculous Apps Installer**.
    <Frame>
      <img 
        src="/images/kb/softaculous-icon.gif" 
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for Dotclear">
    On the left-hand side of the Softaculous interface, type **Dotclear** into the search box. Click on the **Dotclear** link that appears in the search results.
    <Frame>
      <img 
        src="/images/kb/dotclear-softaculous.gif" 
        alt="Searching for Dotclear in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button for Dotclear.
    <Frame>
      <img 
        src="/images/kb/dotclear-install-button.gif" 
        alt="Dotclear Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL</strong>: Select the desired protocol (HTTP or HTTPS) and your domain. If you wish to install Dotclear in a subdirectory (e.g., `example.com/directory`), enter the directory name in the "In Directory" field. Leave it blank to install Dotclear on your main domain (e.g., `example.com`).</li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous general installation screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, enter the following:
    <ul>
      <li><strong>Site Name:</strong> Provide a name for your website, such as "My First Website".</li>
    </ul>
  </Step>
  <Step title="Configure Admin Account">
    Under the **Admin Account** section, enter the following details:
    <ul>
      <li><strong>Admin Username:</strong> Choose a username for your Dotclear admin account. For security reasons, avoid common usernames like "admin" or "administrator".</li>
      <li><strong>Admin Password:</strong> Create a strong and secure password.</li>
      <li><strong>First Name:</strong> Enter your first name.</li>
      <li><strong>Last Name:</strong> Enter your last name.</li>
      <li><strong>Admin Email:</strong> Provide your email address.</li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-gen-install-screen.gif" 
        alt="Softaculous admin account configuration"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>It is highly recommended to copy your admin username and password to a secure location, such as Notepad. You will need these credentials to log in to your Dotclear blog's administration area later.</Note>
  </Step>
  <Step title="Choose Language">
    Select the preferred language for your Dotclear blog from the dropdown menu. The default language is English.
    <Frame>
      <img 
        src="/images/kb/softaculous-choose-language.gif" 
        alt="Softaculous language selection for Dotclear"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    You can expand the **Advanced Options** section if you wish to change the database name or configure automated backups. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to begin the installation process.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous final install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
