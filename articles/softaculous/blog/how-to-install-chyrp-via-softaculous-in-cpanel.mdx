---
title: "Install Chyrp via Softaculous in cPanel"
description: "A guide on how to install Chyrp using the Softaculous Apps Installer in cPanel."
---

Softaculous is a one-click installer that simplifies the process of installing web applications. Follow these steps to install Chyrp on your cPanel account.

<Steps>
  <Step title="Log in to cPanel">
    Access your cPanel account by logging in through your hosting provider's portal.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    Once logged in, scroll down to the **Software** section and click on **Softaculous Apps Installer**.
    <Frame>
      <img 
        src="/images/kb/softaculous-icon.gif" 
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for Chyrp">
    On the left-hand sidebar of the Softaculous interface, type "Chyrp" into the search box. Click on the **Chyrp** link that appears in the search results.
    <Frame>
      <img 
        src="/images/kb/chyrp-softaculous.gif" 
        alt="Searching for Chyrp in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the Chyrp installation process.
    <Frame>
      <img 
        src="/images/kb/chyrp-install-button.gif" 
        alt="Chyrp Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL:</strong> Select the desired protocol (e.g., http:// or https://) and your domain. If you wish to install Chyrp in a subdirectory (e.g., `example.com/directory`), enter the directory name in the "In Directory" field. Leave it blank to install Chyrp on your main domain (e.g., `example.com`).</li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Software Setup screen in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide the following information:
    <ul>
      <li><strong>Site Name:</strong> Enter the name for your Chyrp website (e.g., "My First Blog").</li>
      <li><strong>Site Description:</strong> Provide a brief description of your website.</li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter the following details for your Chyrp administrator account:
    <ul>
      <li><strong>Admin Username:</strong> Choose a username for your admin account. For security, avoid common usernames like "admin" or "administrator".</li>
      <li><strong>Admin Password:</strong> Create a strong and secure password.</li>
      <li><strong>Admin Email:</strong> Enter your email address.</li>
    </ul>
    <Note>
      Copy your admin username and password to a secure location, such as Notepad. You will need these credentials to log in to your Chyrp admin area later.
    </Note>
    <Frame>
      <img 
        src="/images/kb/softaculous-screen2-general.gif" 
        alt="Admin Account setup screen in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure settings like the database name or set up automated backups. You can leave these as default unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the Chyrp installation.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Final Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
