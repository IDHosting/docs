---
title: "Install Open Real Estate via Softaculous in cPanel"
description: "A guide on how to easily install Open Real Estate using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to install Open Real Estate on your hosting account.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel dashboard.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel, click on **Softaculous Apps Installer**.
    <Frame>
      <img 
        src="/images/kb/softaculous-icon.gif" 
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Find Open Real Estate">
    Once Softaculous loads, use the search box on the left-hand side to type "Open Real Estate". Click on the **Open Real Estate** link that appears in the search results.
    <Frame>
      <img 
        src="/images/kb/openrealestate-softaculous.gif" 
        alt="Searching for Open Real Estate in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Begin Installation">
    Click the **Install** button to start the Open Real Estate installation process.
    <Frame>
      <img 
        src="/images/kb/openrealestate-install-button.gif" 
        alt="Open Real Estate Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the required details under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the desired protocol (e.g., http:// or https://) and your domain. If you wish to install Open Real Estate in a subdirectory (e.g., `example.com/directory`), enter the directory name in the "<strong>In Directory</strong>" field. Leave it blank to install in the root of your domain (e.g., `example.com`).
      </li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous Software Setup - Installation URL"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the <strong>Site Settings</strong> section, provide the following:
    <ul>
      <li>
        <strong>Site Name:</strong> Enter a name for your website (e.g., "My Real Estate Listings").
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    In the <strong>Admin Account</strong> section, enter:
    <ul>
      <li>
        <strong>Admin Email:</strong> Your primary email address.
      </li>
      <li>
        <strong>Admin Password:</strong> Choose a strong and secure password.
      </li>
      <li>
        <strong>Admin Username:</strong> Create a username for your Open Real Estate admin panel.
      </li>
    </ul>
    <Note>
      It is highly recommended to copy and save your admin username and password to a secure location, such as Notepad. You will need these credentials to log in to your Open Real Estate admin area later.
    </Note>
    <Frame>
      <img 
        src="/images/kb/softaculous-gen-install-screen-two.gif" 
        alt="Softaculous Admin Account Setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The <strong>Advanced Options</strong> section allows for further customization, such as changing the database name or configuring automated backups.
    <Warning>
      Be mindful that frequent automated backups can consume significant disk space.
    </Warning>
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
