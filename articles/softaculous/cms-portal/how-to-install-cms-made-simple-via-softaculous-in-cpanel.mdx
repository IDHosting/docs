---
title: "Install CMS Made Simple via Softaculous in cPanel"
description: "A guide on how to install CMS Made Simple using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to install CMS Made Simple on your hosting account.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel dashboard.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel, click on **Softaculous Apps Installer**.
    <Frame>
      <img 
        src="/images/kb/softaculous-icon.gif" 
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for CMS Made Simple">
    On the left-hand sidebar, type "CMS Made Simple" into the search box. Click on the **CMS Made Simple** link that appears.
    <Frame>
      <img 
        src="/images/kb/cmsmadesimple-softaculous.gif" 
        alt="Searching for CMS Made Simple in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img 
        src="/images/kb/cmsmadesimple-install-button.gif" 
        alt="CMS Made Simple Install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the appropriate protocol (e.g., http:// or https://) and your domain. In the "<strong>In Directory</strong>" field, enter a directory name if you wish to install CMS Made Simple in a subfolder (e.g., `example.com/directory`). Leave this field blank to install it directly on your main domain (e.g., `example.com`).
      </li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous general installation screen for Software Setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the <strong>Site Settings</strong> section, enter the following:
    <ul>
      <li>
        <strong>Site Name:</strong> Provide a name for your website (e.g., "My CMS Made Simple Site").
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the <strong>Admin Account</strong> section, enter the following details for your CMS Made Simple administrator:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a secure username. Avoid common usernames like "admin" or "administrator".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and complex password.
      </li>
      <li>
        <strong>Admin Email:</strong> Enter your primary email address.
      </li>
    </ul>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as a text file, as you will need them to log in to your CMS Made Simple admin area later.
    </Note>
    <Frame>
      <img 
        src="/images/kb/softaculous-gen-install-screen-two.gif" 
        alt="Softaculous admin account setup screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Language">
    Select the desired language for your CMS Made Simple installation. The default language is English.
    <Frame>
      <img 
        src="/images/kb/softaculous-choose-language.gif" 
        alt="Softaculous language selection screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    You can expand the <strong>Advanced Options</strong> if you wish to customize the database name or configure automated backups. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to begin the installation process.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous final install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
