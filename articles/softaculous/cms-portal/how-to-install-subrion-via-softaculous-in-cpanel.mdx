---
title: "Install Subrion via Softaculous in cPanel"
description: "A guide on how to install Subrion using the Softaculous Apps Installer in cPanel."
---

Softaculous is a one-click installer that simplifies the process of installing web applications. Follow these steps to install Subrion on your cPanel account.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account through your hosting provider's portal.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel dashboard, click on **Softaculous Apps Installer**.
    <Frame>
      <img 
        src="/images/kb/softaculous-icon.gif" 
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Find Subrion">
    Once redirected to the Softaculous interface, use the search box on the left-hand side to type "Subrion". Click on the **Subrion** link that appears in the search results.
    <Frame>
      <img 
        src="/images/kb/subrion-softaculous.gif" 
        alt="Searching for Subrion in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img 
        src="/images/kb/subrion-install-button.gif" 
        alt="Subrion Install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the desired protocol (e.g., http:// or https://) and your domain. If you wish to install Subrion in a subdirectory (e.g., `example.com/directory`), enter the directory name in the "In Directory" field. Leave it blank to install in the root of your domain (e.g., `example.com`).
      </li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous general install screen for URL configuration"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, provide the following details:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a username for your Subrion admin panel. For security, avoid common usernames like "administrator" or "admin".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and secure password.
      </li>
      <li>
        <strong>Admin Email:</strong> Enter your valid email address.
      </li>
    </ul>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as a text file, as you will need them to log in to your Subrion admin area later.
    </Note>
    <Frame>
      <img 
        src="/images/kb/softaculous-general-admin-fields.gif" 
        alt="Softaculous admin account fields"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure settings such as the database name or set up automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to begin the installation process.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Final Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
