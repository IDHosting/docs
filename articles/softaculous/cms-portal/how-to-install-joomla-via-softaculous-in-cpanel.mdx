---
title: "Install Joomla via Softaculous in cPanel"
description: "A guide on how to install Joomla using the Softaculous Apps Installer in cPanel."
---

Softaculous is a one-click installer that simplifies the process of installing web applications. Follow these steps to install Joomla on your hosting account using Softaculous.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account through your hosting provider's portal.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel dashboard, click on **Softaculous Apps Installer**.
    <Frame>
      <img 
        src="/images/kb/softaculous-icon.gif" 
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for Joomla">
    On the left-hand sidebar, type "Joomla" into the search box. Click on the **Joomla** link that appears in the search results.
    <Frame>
      <img 
        src="/images/kb/joomla-softaculous.gif" 
        alt="Searching for Joomla in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img 
        src="/images/kb/joomla-install-button.gif" 
        alt="Joomla Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL:</strong> Select the desired protocol (e.g., `http://` or `https://`) and your domain. In the "In Directory" field, enter a directory name if you want to install Joomla in a subdirectory (e.g., `example.com/directory`). Leave it blank to install in the root of your domain (e.g., `example.com`).</li>
      <li><strong>Choose the version you want to install:</strong> You can typically leave this set to the latest available version.</li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-default-install-screen.gif" 
        alt="Softaculous default installation screen for Joomla"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide the following details:
    <ul>
      <li><strong>Site Name:</strong> Enter the name for your website (e.g., "My First Website").</li>
      <li><strong>Site Description:</strong> Enter a tagline or slogan for your website.</li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter:
    <ul>
      <li><strong>Admin Username:</strong> Choose a secure username for your Joomla administrator account. Avoid common usernames like "admin" or "administrator".</li>
      <li><strong>Admin Password:</strong> Create a strong and complex password.</li>
      <li><strong>Real Name:</strong> You can use your full name or leave it as is.</li>
      <li><strong>Admin Email:</strong> Enter a valid email address for administrative purposes.</li>
    </ul>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location (like Notepad) immediately after setting them. You will need these credentials to log in to your Joomla admin area later.
    </Note>
    <Frame>
      <img 
        src="/images/kb/softaculous-install-screen2.gif" 
        alt="Admin account setup screen in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Language">
    Select the preferred language for your Joomla installation from the dropdown menu. The default language is English.
    <Frame>
      <img 
        src="/images/kb/softaculous-choose-language.gif" 
        alt="Language selection for Joomla installation"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    You can expand the <strong>Advanced Options</strong> section if you wish to configure advanced settings such as changing the database name or setting up automated backups. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to begin the Joomla installation process.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Final Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

Once the installation is complete, you will be provided with links to your new Joomla website and its administrative area.
