---
title: "Install MODX via Softaculous in cPanel"
description: "A guide on how to install MODX using the Softaculous Apps Installer in cPanel."
---

Softaculous is a one-click installer that simplifies the process of installing web applications. Follow these steps to install MODX on your hosting account using Softaculous.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account through your hosting provider's portal.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel dashboard, click on **Softaculous Apps Installer**.
    <Frame>
      <img src="/images/kb/softaculous-icon.gif" alt="Softaculous Apps Installer Icon" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for MODX">
    On the left-hand sidebar, type `MODX` into the search box. Click on the **MODX** link that appears in the search results.
    <Frame>
      <img src="/images/kb/modx-softaculous.gif" alt="MODX search result in Softaculous" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img src="/images/kb/modx-install-button.gif" alt="MODX Install Button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL:</strong> Select the desired protocol (e.g., http:// or https://) and your domain. In the <strong>In Directory</strong> field, enter a directory name if you want to install MODX in a subdirectory (e.g., `example.com/directory`). Leave this field blank to install MODX on your main domain (e.g., `example.com`).</li>
    </ul>
    <Frame>
      <img src="/images/kb/softaculous-general-install-screen.gif" alt="Softaculous Software Setup Screen" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the <strong>Site Settings</strong> section, enter the following:
    <ul>
      <li><strong>Site Name:</strong> Provide a name for your website (e.g., `My MODX Site`).</li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the <strong>Admin Account</strong> section, enter the following details:
    <ul>
      <li><strong>Admin Username:</strong> Choose a username for your MODX administrator account. For security, avoid common usernames like `admin` or `administrator`.</li>
      <li><strong>Admin Password:</strong> Create a strong and secure password.</li>
      <li><strong>Admin Email:</strong> Enter your email address.</li>
    </ul>
    <Note>
      Copy your admin username and password to a secure location, such as Notepad. You will need these credentials to log in to your MODX admin area later.
    </Note>
    <Frame>
      <img src="/images/kb/softaculous-gen-install-screen-two.gif" alt="Softaculous Admin Account Setup" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    You can configure <strong>Advanced Options</strong> if you wish to change the database name or set up automated backups. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button.
    <Frame>
      <img src="/images/kb/softaculous-install-button.gif" alt="Softaculous Install Button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
</Steps>

Once the installation is complete, you will be provided with the links to your MODX website and its administration area.
