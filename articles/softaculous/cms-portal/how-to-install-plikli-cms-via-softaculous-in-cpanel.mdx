---
title: "Install Plikli CMS via Softaculous in cPanel"
description: "A guide on how to install Plikli CMS using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to install Plikli CMS with ease.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account through your hosting provider.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel dashboard, click on **Softaculous Apps Installer**.

    <Frame>
      <img src="/images/kb/softaculous-icon.gif" alt="Softaculous Apps Installer icon" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    You will be redirected to the Softaculous interface.
  </Step>
  <Step title="Search for Plikli CMS">
    On the left-hand sidebar, type **Plikli CMS** into the search box. Click on the **Plikli CMS** link that appears in the search results.

    <Frame>
      <img src="/images/kb/pliklicms-softaculous.gif" alt="Searching for Plikli CMS in Softaculous" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the Plikli CMS installation process.

    <Frame>
      <img src="/images/kb/pliklicms-install-button.gif" alt="Plikli CMS Install button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    *   **Choose Installation URL:** Select the appropriate protocol (e.g., `http://` or `https://`) and your domain name.
        *   **In Directory:** If you wish to install Plikli CMS in a subdirectory (e.g., `example.com/directory`), enter the directory name here. Leave this field blank to install Plikli CMS on your main domain (e.g., `example.com`).

    <Frame>
      <img src="/images/kb/softaculous-general-install-screen.gif" alt="Softaculous Software Setup screen" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide the following information:
    *   **Site Name:** Enter the name for your website (e.g., **Plikli CMS**).
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter your administrator credentials:
    *   **Admin Username:** Choose a username for your Plikli CMS administrator. For security reasons, avoid using generic usernames like `administrator` or `admin`.
    *   **Admin Password:** Create a strong and secure password.
    *   **Admin Email:** Enter your email address.

    <Frame>
      <img src="/images/kb/softaculous-gen-install-screen-two.gif" alt="Softaculous Admin Account setup" style={{ width: '100%', height: 'auto' }} />
    </Frame>

    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as Notepad, as you will need them to access the Plikli CMS admin area later.
    </Note>
  </Step>
  <Step title="Choose Language">
    Select the desired language for your Plikli CMS installation. The default language is English.

    <Frame>
      <img src="/images/kb/softaculous-choose-language.gif" alt="Softaculous language selection" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database settings or set up automated backups. You can typically leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation.

    <Frame>
      <img src="/images/kb/softaculous-install-button.gif" alt="Softaculous final install button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
</Steps>
