---
title: "Install WebsiteBaker via Softaculous in cPanel"
description: "A guide on how to install WebsiteBaker using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to install WebsiteBaker on your hosting account.

<Steps>
  <Step title="Log in to your cPanel account">
    Begin by accessing your cPanel dashboard.
  </Step>
  <Step title="Access Softaculous Apps Installer">
    Navigate to the **Software** section and click on **Softaculous Apps Installer**.

    <Frame>
      <img
        src="/images/kb/softaculous-icon.gif"
        alt="Softaculous Apps Installer Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Find WebsiteBaker">
    Once redirected to the Softaculous interface, use the search box on the left-hand side to type "WebsiteBaker". Click on the **WebsiteBaker** link that appears in the search results.

    <Frame>
      <img
        src="/images/kb/websitebaker-softaculous.gif"
        alt="Searching for WebsiteBaker in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the WebsiteBaker installation process.

    <Frame>
      <img
        src="/images/kb/websitebaker-install-button.gif"
        alt="WebsiteBaker Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    In the **Software Setup** section, configure the following:
    *   **Choose Installation URL:** Select the desired protocol (HTTP or HTTPS) and domain. If you wish to install WebsiteBaker in a subdirectory (e.g., `example.com/directory`), enter the directory name in the **In Directory** field. Leave it blank to install in the root of your domain (e.g., `example.com`).

    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Softaculous General Install Screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide:
    *   **Site Name:** Enter the name for your website.
    *   **Site Description:** Add a brief description of your website.
  </Step>
  <Step title="Set Up Admin Account">
    In the **Admin Account** section, enter:
    *   **Admin Username:** Choose a secure username for your WebsiteBaker admin panel. Avoid common usernames like "admin" or "administrator".
    *   **Admin Password:** Create a strong and complex password.
    *   **Admin Email:** Provide a valid email address.

    <Frame>
      <img
        src="/images/kb/softaculous-screen2-general.gif"
        alt="Softaculous Admin Account Settings"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Language">
    Select the preferred language for your WebsiteBaker installation. The default is English.

    <Frame>
      <img
        src="/images/kb/softaculous-choose-language.gif"
        alt="Softaculous Choose Language"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>

    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as Notepad, for future reference. You will need these credentials to log in to your WebsiteBaker admin area.
    </Note>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows for further customization, such as changing the database name or configuring automated backups. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the WebsiteBaker installation.

    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
