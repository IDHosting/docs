---
title: "Install PHP-Fusion via Softaculous in cPanel"
description: "A guide on how to easily install PHP-Fusion using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to install PHP-Fusion on your website.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account through your hosting provider.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel dashboard, click on **Softaculous Apps Installer**.
    <Frame>
      <img
        src="/images/kb/softaculous-icon.gif"
        alt="Softaculous Apps Installer Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for PHP-Fusion">
    On the left-hand sidebar, type `PHP-Fusion` into the search box. Click on the **PHP-Fusion** link that appears.
    <Frame>
      <img
        src="/images/kb/php-fusion-softaculous.gif"
        alt="Searching for PHP-Fusion in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img
        src="/images/kb/php-fusion-install-button.gif"
        alt="PHP-Fusion Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the desired protocol (e.g., http:// or https://) and your domain. In the "<strong>In Directory</strong>" field, enter a directory name if you wish to install PHP-Fusion in a subdirectory (e.g., `example.com/directory`). Leave this field blank to install in the root of your domain (e.g., `example.com`).
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Softaculous General Install Screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, enter the following:
    <ul>
      <li>
        <strong>Site Name:</strong> The name of your website (e.g., `PHP-Fusion`).
      </li>
      <li>
        <strong>Site Email:</strong> Your primary email address.
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, provide the following details:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a secure username for your PHP-Fusion administrator account. Avoid common usernames like "admin" or "administrator".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and unique password.
      </li>
      <li>
        <strong>Admin Email:</strong> Your email address for administrative purposes.
      </li>
      <li>
        <strong>User Password:</strong> This field seems to be a duplicate or mislabeled in the original HTML. Ensure you are setting a strong password for the admin account.
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/php-fusion-install-screen2.gif"
        alt="PHP-Fusion Installation Screen - Admin Details"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location (like a password manager or a text file) immediately after setting them. You will need these credentials to log in to your PHP-Fusion admin area later.
    </Note>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database settings or set up automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent automated backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button.
    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
