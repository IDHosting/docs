---
title: "Install Fork via Softaculous in cPanel"
description: "A guide on how to install Fork CMS using the Softaculous Apps Installer in cPanel."
---

Softaculous is a one-click installer that simplifies the process of installing web applications. Follow these steps to install Fork CMS on your hosting account.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel dashboard.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel, click on **Softaculous Apps Installer**.
    <Frame>
      <img
        src="/images/kb/softaculous-icon.gif"
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    You will be redirected to the Softaculous interface.
  </Step>
  <Step title="Search for Fork CMS">
    On the left-hand sidebar, type "Fork" into the search box. Click on the **Fork** link that appears.
    <Frame>
      <img
        src="/images/kb/fork-softaculous.gif"
        alt="Searching for Fork CMS in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img
        src="/images/kb/fork-install-button.gif"
        alt="Fork CMS Install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the desired protocol (e.g., http:// or https://) and your domain. In the "<strong>In Directory</strong>" field, enter a directory name if you want to install Fork in a subdirectory (e.g., `example.com/directory`). Leave it blank to install in the root of your domain (e.g., `example.com`).
        <Frame>
          <img
            src="/images/kb/softaculous-general-install-screen.gif"
            alt="Softaculous general installation screen for URL configuration"
            style={{ width: '100%', height: 'auto' }}
          />
        </Frame>
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the <strong>Admin Account</strong> section, enter the following details:
    <ul>
      <li>
        <strong>Admin Email:</strong> Your email address.
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong password.
        <Frame>
          <img
            src="/images/kb/general-admin-install-screen.gif"
            alt="Softaculous admin account setup screen"
            style={{ width: '100%', height: 'auto' }}
          />
        </Frame>
        <Note>
          Copy your admin email and password to a safe place, such as Notepad. You will need these credentials to log in to your Fork CMS admin area later.
        </Note>
      </li>
    </ul>
  </Step>
  <Step title="Advanced Options (Optional)">
    The <strong>Advanced Options</strong> section allows you to configure database names or set up automated backups. You can leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
    <br />
    Finally, scroll to the bottom of the page and click the **Install** button.
    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous final install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
