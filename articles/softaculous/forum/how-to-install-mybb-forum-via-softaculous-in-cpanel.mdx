---
title: "Install MyBB Forum via Softaculous in cPanel"
description: "A guide on how to install MyBB Forum using the Softaculous Apps Installer in cPanel."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. Follow these steps to easily install MyBB Forum on your cPanel account.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account by navigating to your cPanel login URL and entering your credentials.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel dashboard, click on **Softaculous Apps Installer**.

    <Frame>
      <img 
        src="/images/kb/softaculous-icon.gif" 
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    You will be redirected to the Softaculous interface.
  </Step>
  <Step title="Search for MyBB">
    On the left-hand side of the Softaculous interface, use the search box to type **MyBB**. Click on the **MyBB** link that appears in the search results.

    <Frame>
      <img 
        src="/images/kb/mybb-softaculous.gif" 
        alt="Searching for MyBB in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the MyBB installation process.

    <Frame>
      <img 
        src="/images/kb/mybb-install-button.gif" 
        alt="MyBB Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Under the **Software Setup** section, fill in the following details:
    *   **Choose Installation URL:** Select the appropriate protocol (HTTP or HTTPS) and your domain. If you wish to install MyBB in a subdirectory (e.g., `example.com/forum`), enter the directory name in the "**In Directory**" field. Leave it blank to install MyBB on your main domain (e.g., `example.com`).

    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="MyBB Software Setup configuration screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Board Settings">
    In the **Board Settings** section, provide the following information:
    *   **Site Name:** The overall name of your website (e.g., "My Community").
    *   **Board Name:** The specific name for your MyBB forum.
    *   **Board E-mail:** The primary email address for your forum.
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter:
    *   **Admin Username:** Your desired username for the MyBB administrator account. For security, avoid common usernames like "admin" or "administrator".
    *   **Admin Password:** Create a strong and secure password for your admin account.
    *   **Admin Email:** Your email address. This is crucial for password recovery and important notifications.

    <Note>
      It is highly recommended to copy your admin username and password to a secure location like Notepad. You will need these credentials to access the MyBB admin control panel later.
    </Note>

    <Frame>
      <img 
        src="/images/kb/mybb-install-screen.gif" 
        alt="MyBB Admin Account setup screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Language">
    Select the desired language for your MyBB forum from the **Choose Language** dropdown. The default is English.

    <Frame>
      <img 
        src="/images/kb/softaculous-choose-language.gif" 
        alt="MyBB Language selection screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database details or set up automated backups. You can typically leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the MyBB forum installation.

    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Final Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
