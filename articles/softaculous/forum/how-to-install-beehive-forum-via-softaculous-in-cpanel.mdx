---
title: "Install Beehive Forum via Softaculous in cPanel"
description: "A guide on how to install Beehive Forum using the Softaculous Apps Installer in cPanel."
---

Softaculous is a one-click installer that simplifies the process of setting up web applications. Follow these steps to install Beehive Forum on your cPanel account.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel dashboard.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel, click on **Softaculous Apps Installer**.
    <Frame>
      <img
        src="/images/kb/softaculous-icon.gif"
        alt="Softaculous Apps Installer icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Search for Beehive">
    Once redirected to the Softaculous interface, use the search box on the left-hand side to find "Beehive". Click on the Beehive link when it appears.
    <Frame>
      <img
        src="/images/kb/beehive-softaculous.gif"
        alt="Searching for Beehive in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the Beehive installation process.
    <Frame>
      <img
        src="/images/kb/beehive-install-button.gif"
        alt="Beehive Install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary details under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the appropriate protocol (e.g., http, https) and domain. In the "In Directory" field, enter a directory name if you wish to install Beehive in a subdirectory (e.g., `example.com/forum`). Leave it blank to install on your main domain (e.g., `example.com`).
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Software Setup configuration in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Enter the following details under the **Site Settings** section:
    <ul>
      <li>
        <strong>Default Forum Webtag:</strong> Provide a unique tag for your forum, such as `MyBeehive_Forum`.
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter the following information:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a username for your Beehive administrator. For security reasons, avoid using common usernames like "administrator" or "admin".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and secure password.
      </li>
      <li>
        <strong>Admin Email:</strong> Enter your email address. This is crucial for password recovery and receiving notifications.
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/beehive-install-screen.gif"
        alt="Admin Account details for Beehive installation"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      It is highly recommended to copy and save your admin username and password to a secure location, such as Notepad. You will need these credentials to access the Beehive admin area later.
    </Note>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database settings or set up automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation.
    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous Install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
