---
title: "Install Advanced Electron Forum (AEF) via Softaculous in cPanel"
description: "A guide on how to install Advanced Electron Forum (AEF) using the Softaculous Apps Installer in cPanel."
---

Softaculous makes installing web applications straightforward. Follow these steps to install Advanced Electron Forum (AEF).

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account through your hosting provider's portal.
  </Step>
  <Step title="Navigate to Softaculous Apps Installer">
    In the **Software** section of your cPanel dashboard, click on **Softaculous Apps Installer**.
    <Frame>
      <img src="/images/kb/softaculous-icon.gif" alt="Softaculous Apps Installer icon" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    cPanel will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for AEF">
    On the left-hand side of the Softaculous interface, type `AEF` into the search box. Click on the **AEF** link that appears in the search results.
    <Frame>
      <img src="/images/kb/aef-softaculous.gif" alt="AEF search result in Softaculous" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img src="/images/kb/aef-install-button.gif" alt="AEF Install button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL:</strong> Select the appropriate protocol (e.g., http, https) and domain. In the <strong>In Directory</strong> field, enter a directory name if you wish to install AEF in a subdirectory (e.g., `example.com/forum`). Leave this field blank to install AEF in the root of your domain (e.g., `example.com`).</li>
    </ul>
    <Frame>
      <img src="/images/kb/softaculous-general-install-screen.gif" alt="Softaculous general installation screen" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Board Settings">
    Under the <strong>Board Settings</strong> section, enter the following details:
    <ul>
      <li><strong>Board Name:</strong> The name of your forum.</li>
      <li><strong>Board E-mail:</strong> The primary email address for your forum.</li>
      <li><strong>Use UTF-8 Character Set:</strong> Select this option if you plan to use multiple languages in your forum.</li>
    </ul>
  </Step>
  <Step title="Configure Admin Account">
    Under the <strong>Admin Account</strong> section, provide the following:
    <ul>
      <li><strong>Admin Username:</strong> Your desired username for the AEF administrator account. For security reasons, avoid using generic usernames like `administrator` or `admin`.</li>
      <li><strong>Admin Password:</strong> Create a strong and secure password for your admin account.</li>
      <li><strong>Admin Email:</strong> Your email address. This is crucial for password recovery and receiving important notifications.</li>
    </ul>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as a text file, immediately after setting them. You will need these credentials to access the admin area of your AEF forum.
    </Note>
    <Frame>
      <img src="/images/kb/aef-install-screen.gif" alt="AEF installation screen with admin details" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The <strong>Advanced Options</strong> section allows you to configure database settings or set up automated backups. You can typically leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to begin the installation process.
    <Frame>
      <img src="/images/kb/softaculous-install-button.gif" alt="Softaculous Install button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
</Steps>
