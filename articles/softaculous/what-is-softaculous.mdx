---
title: "What is Softaculous"
description: "An overview of Softaculous, an auto-installer for web applications."
---

Softaculous is an auto-installer that allows for the automated installation of various web scripts with a single click. It supports popular scripts and facilitates hassle-free upgrades of web applications.

You can install a wide range of scripts through Softaculous, including:
*   WordPress
*   Joomla
*   Drupal
*   MyBB Forum
*   SMF
*   Mambo
*   And many others

Softaculous simplifies common tasks such as installing scripts, backing up and restoring them, and uninstalling software, all with a single click. It also features an automatic backup creation capability.

<Info>
Softaculous is often integrated into hosting control panels like cPanel and Plesk, providing users with a convenient way to manage their web applications.
</Info>
