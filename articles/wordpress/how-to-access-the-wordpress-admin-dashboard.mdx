---
title: "Access the WordPress Admin Dashboard"
description: "Learn how to log in to your WordPress admin dashboard to manage your website."
---

WordPress is a user-friendly Content Management System (CMS). Follow these steps to access your WordPress admin dashboard.

<Steps>
  <Step title="Navigate to the Login Page">
    Open your web browser and go to `yourdomain.com/wp-admin`.
    <Note>Remember to replace `yourdomain.com` with your actual domain name.</Note>
  </Step>
  <Step title="Enter Credentials and Login">
    Enter your WordPress username and password in the provided fields. Then, click the **Login** button.
    <Frame>
      <img
        src="/images/kb/wordpress-login-dashboard.gif"
        alt="WordPress login screen with fields for username and password"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Dashboard">
    If your credentials are correct, you will be successfully logged into the WordPress Admin Dashboard.
    <Frame>
      <img
        src="/images/kb/wordpress-admin-dashboard.gif"
        alt="WordPress Admin Dashboard interface"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

From the WordPress admin dashboard, you can manage your site's content, customize themes, install plugins, and perform various other administrative tasks.
