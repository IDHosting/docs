---
title: "Remove a Backup File Using DirectAdmin"
description: "Learn how to remove backup files from your DirectAdmin account to free up disk space."
---

You can remove backup files from your DirectAdmin to reduce disk space usage. If you are getting an error while generating a backup, such as indicating that the backup exceeds the storage limit, it will be useful for you to remove it.

<Steps>
  <Step title="Log in to your DirectAdmin Account">
    Access your DirectAdmin control panel by logging in with your credentials.
  </Step>
  <Step title="Open File Manager">
    In the **System Info & Files** section, click on **File Manager**. Alternatively, type **File Manager** in the **search bar**, and the option will appear. Click on it.
    <Frame>
      <img 
        src="/images/kb/directadmin-filemanager-icon.gif" 
        alt="DirectAdmin File Manager Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Navigate to the backups directory">
    Open the <strong>backups</strong> directory.
    <Frame>
      <img 
        src="/images/kb/directadmin-click-backupsfldr.gif" 
        alt="DirectAdmin Click Backups Folder"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Remove the backup file">
    Right-click on the backup file you wish to remove and choose <strong>Remove</strong> from the context menu.
    <Frame>
      <img 
        src="/images/kb/remove-backup-file.gif" 
        alt="DirectAdmin Remove Backup File Context Menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Confirm removal">
    Click <strong>Remove</strong> again to confirm the deletion of the backup file.
  </Step>
</Steps>

Congratulations. You have successfully removed the backup file.
