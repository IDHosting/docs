---
title: "Create a Domain Alias in DirectAdmin"
description: "Learn how to create a domain alias in DirectAdmin to redirect one domain to another."
---

A domain alias allows you to access one domain through another. For example, you can set up `old.example.com` to point to `new.example.com`. This guide will walk you through the process in DirectAdmin.

<Steps>
  <Step title="Log in to DirectAdmin">
    Access your DirectAdmin account using your credentials.
  </Step>
  <Step title="Navigate to Domain Pointers">
    In the **Account Manager** section, click on **Domain Pointers**.
    <br />
    Alternatively, you can use the search bar: type **Domain Pointers** and select the option when it appears.
    <Frame>
      <img
        src="/images/kb/da-domainpointer-icon.gif"
        alt="DirectAdmin Domain Pointers icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Create a New Domain Pointer">
    Click on the **Create Domain Pointer** button, typically located in the top right corner of the page.
    <Frame>
      <img
        src="/images/kb/da-createdomainpointer-btn.gif"
        alt="DirectAdmin Create Domain Pointer button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Enter Source Domain">
    In the **Source Domain** field, enter the domain name you want to use as the alias (e.g., `old.example.com`).
    <Frame>
      <img
        src="/images/kb/da-domainpointer-createalias.gif"
        alt="DirectAdmin Create Domain Pointer alias input"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Enable Alias Creation">
    Ensure that the **Create as an Alias** option is checked.
  </Step>
  <Step title="Confirm Creation">
    Click the **Create** button to finalize the domain alias setup.
  </Step>
</Steps>

Your domain alias will now be successfully created and active.
