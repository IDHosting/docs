---
title: "Change an FTP Account Password in DirectAdmin"
description: "Learn how to change the password for your FTP accounts using the DirectAdmin control panel."
---

This guide will walk you through the process of changing the password for a specific FTP account in DirectAdmin. If you need to change the password for the main root FTP account, you will need to change your DirectAdmin account password.

<Steps>
  <Step title="Log in to DirectAdmin">
    Access your DirectAdmin control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to FTP Management">
    In the **Account Manager** section, click on **FTP Management**.
    Alternatively, you can use the **search bar** at the top of the page, type "FTP Management", and select the option when it appears.
    <Frame>
      <img 
        src="/images/kb/da-ftpmanagement-icon.gif" 
        alt="DirectAdmin FTP Management Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select the FTP Account">
    Under the **FTP Accounts** list, find and select the specific FTP account for which you wish to change the password.
    <Frame>
      <img 
        src="/images/kb/da-click-on-ftpuser.gif" 
        alt="DirectAdmin Click on FTP User"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Enter New Password">
    In the **Enter Password** field, type your desired new password. For a strong, randomly generated password, click the **Generate Random Password** button.
    <Note>
      Ensure you copy the generated password to a secure location immediately after creation.
    </Note>
    <Frame>
      <img 
        src="/images/kb/da-ftpuserpass-change.gif" 
        alt="DirectAdmin FTP User Password Change Field"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Confirm Changes">
    Click the **Modify** button to save your changes.
  </Step>
</Steps>

Your FTP account password has now been successfully changed.
