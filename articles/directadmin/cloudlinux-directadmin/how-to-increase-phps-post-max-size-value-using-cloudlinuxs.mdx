---
title: "Increase PHP's post_max_size Value Using CloudLinux's Selector in DirectAdmin"
description: "Learn how to adjust the post_max_size value for your PHP installation using the CloudLinux PHP Selector within the DirectAdmin control panel."
---

You can modify PHP's `post_max_size` value to suit your needs using the CloudLinux PHP Selector in DirectAdmin.

<Steps>
  <Step title="Log in to your DirectAdmin account">
    Access your DirectAdmin control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to Select PHP Version">
    In the **Extra Features** section, click on **Select PHP version**. Alternatively, you can type "Select PHP" into the navigation filter box, and the option should appear. Click on it.
    <Frame>
      <img 
        src="/images/kb/da-cl-php-selector-icon.gif" 
        alt="DirectAdmin CloudLinux PHP Selector Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Options Menu">
    Click on the **Options** menu.
    <Frame>
      <img 
        src="/images/kb/cl-options-menu.gif" 
        alt="CloudLinux Options Menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>If the **Options** page displays an error related to the native PHP version, you must first change it to a non-native version before proceeding.</Note>
  </Step>
  <Step title="Adjust post_max_size">
    Locate the **post_max_size** setting. Click on the drop-down menu next to it and select the desired limit according to your requirements.
    <Frame>
      <img 
        src="/images/kb/cl-post_max_size.gif" 
        alt="CloudLinux post_max_size setting"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    A confirmation message will appear, indicating that the Maximum Post Size has been successfully changed.
  </Step>
</Steps>
