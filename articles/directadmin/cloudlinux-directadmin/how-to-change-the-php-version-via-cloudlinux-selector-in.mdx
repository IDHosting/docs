---
title: "Change the PHP Version via CloudLinux Selector in DirectAdmin"
description: "Learn how to change your website's PHP version using the CloudLinux Selector in DirectAdmin."
---

You can use CloudLinux's Select PHP Version tool to easily change your website's PHP version within your DirectAdmin control panel. This guide will walk you through the process.

<Steps>
  <Step title="Log in to DirectAdmin">
    Begin by logging into your DirectAdmin account.
  </Step>
  <Step title="Access Select PHP Version">
    Navigate to the **Extra Features** section. Here, you will find an option labeled **Select PHP version**. Alternatively, you can type "Select PHP" into the navigation filter box, and the option should appear. Click on it to proceed.
    <Frame>
      <img 
        src="/images/kb/da-cl-php-selector-icon.gif" 
        alt="DirectAdmin CloudLinux PHP Selector Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Your PHP Version">
    On the right side of the **Current PHP version** label, you'll see a drop-down menu. Click on this menu to reveal the available PHP versions. Select the version you wish to use for your website.
    <Frame>
      <img 
        src="/images/kb/plesk-da-cl-change-php-version.gif" 
        alt="DirectAdmin CloudLinux Change PHP Version Dropdown"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Apply Changes">
    After selecting your desired PHP version, click the **Apply** button to save your changes.
    <Frame>
      <img 
        src="/images/kb/cl-set-as-php-version.gif" 
        alt="CloudLinux Set as PHP Version Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

You have now successfully changed your website's PHP version using the CloudLinux Selector in DirectAdmin.
