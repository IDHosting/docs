---
title: "Increase or Decrease PHP Memory Limit via CloudLinux Selector in DirectAdmin"
description: "Learn how to adjust your PHP memory limit using the CloudLinux PHP Selector in DirectAdmin."
---

You can modify the **PHP Memory Limit** as per your needs using the CloudLinux PHP Selector in DirectAdmin.

<Steps>
  <Step title="Log in to your DirectAdmin account">
    Access your DirectAdmin control panel.
  </Step>
  <Step title="Navigate to Select PHP version">
    In the **Extra Features** section, click on **Select PHP version**. Alternatively, you can type **Select PHP** in the navigation filter box, and the option should appear.
    <Frame>
      <img 
        src="/images/kb/da-cl-php-selector-icon.gif" 
        alt="DirectAdmin CloudLinux PHP Selector Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access PHP Options">
    Click on the **Options** menu.
    <Frame>
      <img 
        src="/images/kb/cl-options-menu.gif" 
        alt="CloudLinux Options Menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      If the **Options** page displays an error related to the native PHP version, you must first change it to a non-native version before proceeding.
    </Note>
  </Step>
  <Step title="Adjust memory_limit">
    Locate the **memory_limit** setting. Click on the drop-down menu next to it and select the desired memory limit that suits your requirements.
    <Frame>
      <img 
        src="/images/kb/cl-increase-decrease-php-memory-limit.gif" 
        alt="CloudLinux Increase or Decrease PHP Memory Limit"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    A confirmation message will appear indicating that the memory limit has been successfully modified.
  </Step>
</Steps>
