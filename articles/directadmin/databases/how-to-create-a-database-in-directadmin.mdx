## Create a Database in DirectAdmin

This tutorial shows you how to create a new database in DirectAdmin.

<Steps>
  <Step title="Log in to your DirectAdmin Account">
    Access your DirectAdmin control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to Databases">
    In the **Account Manager** section, click on **Databases**.
    <br />
    Alternatively, you can use the **search bar** at the top of the page. Type "Databases" and click on the result.
    <Frame>
      <img 
        src="/images/kb/da-databases-icon.gif" 
        alt="DirectAdmin Databases Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Create the Database">
    In the **Database Name** field, enter your desired name for the new database.
    <br />
    Then, click the **Create** button located in the bottom right corner.
    <Frame>
      <img 
        src="/images/kb/da-mysql-dbcr-from.gif" 
        alt="DirectAdmin MySQL Database Creation Form"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      Please copy the newly-created database details (username, password, database name) to a safe place immediately after creation.
    </Note>
  </Step>
</Steps>

You have now successfully created a database in DirectAdmin.
