---
title: "Change a Database Password in DirectAdmin"
description: "This tutorial guides you through the process of changing your database password using the DirectAdmin control panel."
---

This tutorial shows you how to change the password for a database in DirectAdmin.

<Steps>
  <Step title="Log in to DirectAdmin">
    Access your DirectAdmin account by logging in with your credentials.
  </Step>
  <Step title="Navigate to Databases">
    In the **Account Manager** section, click on **Databases**.
    Alternatively, you can type **Databases** in the **search bar** and select the option when it appears.
    <Frame>
      <img
        src="/images/kb/da-databases-icon.gif"
        alt="DirectAdmin Databases Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Manage Users">
    On the right side of the Databases page, click on **Manage Users**.
    <Frame>
      <img
        src="/images/kb/da-databases-manage-users-button.gif"
        alt="DirectAdmin Manage Users Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select User Account">
    From the list of **User Accounts**, click on **Manage** next to the user account associated with your database.
    <Frame>
      <img
        src="/images/kb/da-mysql-db-userlist.gif"
        alt="DirectAdmin MySQL Database User List"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Change Password">
    Under the **Password Management** section, you can either enter a new password manually or click the generate password icon to create a strong, random password.
    Once you have set your desired password, click on **Change Password**.
    <Frame>
      <img
        src="/images/kb/da-mysql-change-pass-option.gif"
        alt="DirectAdmin MySQL Change Password Option"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      Please copy your new database password and store it in a safe place. Losing this password will prevent you from accessing your database.
    </Note>
  </Step>
</Steps>

Your database password has now been successfully changed in DirectAdmin.
