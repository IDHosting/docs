---
title: "Create a New Database User in DirectAdmin"
description: "A guide on how to create a new database user within the DirectAdmin control panel."
---

Learn how to create a new database user in DirectAdmin. Follow the steps below to set it up.

<Steps>
  <Step title="Log in to your DirectAdmin Account">
    Access your DirectAdmin control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to Databases">
    In the **Account Manager** section, click on **Databases**.
    Alternatively, you can type **Databases** in the **search bar** and select the option when it appears.
    <Frame>
      <img 
        src="/images/kb/da-databases-icon.gif" 
        alt="DirectAdmin Databases Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Manage Users">
    On the right side of the Databases page, click on **Manage Users**.
  </Step>
  <Step title="Create New User Details">
    Under the **Create Users** section, enter your desired new username and password.
    If you prefer a randomly generated password, click the **Generate Random Password** icon.
    <Frame>
      <img 
        src="/images/kb/da-create-user-details.gif" 
        alt="DirectAdmin Create User Details"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Confirm Creation">
    Click the **Create** button to finalize the user creation.
  </Step>
</Steps>

<Note>
  Please copy the newly created database user details (username and password) to a safe place immediately after creation.
</Note>
