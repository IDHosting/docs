---
title: "Generate a cPanel Backup and Send it to FTP"
description: "Learn how to create a full cPanel backup and automatically send it to a remote FTP server."
---

Backing up your website data regularly is crucial for security and disaster recovery. This guide will walk you through the process of generating a full cPanel backup and configuring it to be sent directly to a remote FTP server.

## Steps to Generate and Send a cPanel Backup to FTP

<Steps>
  <Step title="Log in to your cPanel Account">
    Access your cPanel account using your provided credentials.
  </Step>
  <Step title="Navigate to the Backup Section">
    Once logged in, locate the **Files** section and click on the **Backup** icon.
    <Frame>
      <img
        src="/images/kb/backup-icon.gif"
        alt="cPanel Backup Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate a Full Website Backup">
    Under the **Full Backup** section, click on the **Download a Full Website Backup** button.
    <Frame>
      <img
        src="/images/kb/download-button-full-backup.gif"
        alt="Download Full Website Backup Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Backup Destination">
    In the **Generate a Full Backup** section, you will need to provide the following details:
    *   **Backup Destination:** Select **Remote FTP Server** from the dropdown menu.
    *   **Email:** Enter your email address to receive notifications about the backup process.
    *   **Remote Server:** Enter the IP address or domain name of your FTP server (e.g., `ftp.example.com`).
    *   **Remote User:** Your FTP username.
    *   **Remote Password:** Your FTP password.
    *   **Port:** The port number for your FTP connection (commonly `21`).
    *   **Remote Dir:** Specify the directory on the FTP server where you want to store the backup (e.g., `/home/username/backupfolder/`).

    <Frame>
      <img
        src="/images/kb/generate-remote-backup-config.gif"
        alt="Generate Full Backup Configuration with Remote FTP Details"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Generate the Backup">
    After filling in all the required FTP details, click on the **Generate Backup** button to start the process.
  </Step>
</Steps>

<Note>
  You will receive an email notification once the backup process is completed. Ensure that the FTP server details are correct to avoid any transfer errors.
</Note>
