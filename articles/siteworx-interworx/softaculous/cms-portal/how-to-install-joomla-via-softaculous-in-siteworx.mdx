---
title: "Install Joomla via Softaculous in SiteWorx"
description: "A guide on how to install Joomla using the Softaculous installer within your SiteWorx control panel."
---

Softaculous is a one-click installer that simplifies the process of adding web applications to your website. This guide will walk you through installing Joomla using Softaculous in your SiteWorx account.

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Access your SiteWorx control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to Softaculous">
    Once logged in, locate and click on **Softaculous** within the **Additional Features** section.
    <Frame>
      <img
        src="/images/kb/sw-softaculous-icon.gif"
        alt="Softaculous icon in SiteWorx"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    SiteWorx will automatically redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for Joomla">
    On the left-hand side of the Softaculous panel, use the search box to type "Joomla". Click on the **Joomla** link that appears in the search results.
    <Frame>
      <img
        src="/images/kb/joomla-softaculous.gif"
        alt="Searching for Joomla in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the Joomla installation process.
    <Frame>
      <img
        src="/images/kb/joomla-install-button.gif"
        alt="Joomla install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the required details under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the desired protocol (HTTP or HTTPS) and domain. In the <strong>In Directory</strong> field, enter a directory name if you wish to install Joomla in a subdirectory (e.g., `example.com/directory`). Leave this field blank to install Joomla in your main domain (e.g., `example.com`).
      </li>
      <li>
        <strong>Choose the version you want to install:</strong> You can typically leave this set to the latest available version.
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/softaculous-default-install-screen.gif"
        alt="Softaculous default installation screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide the following information:
    <ul>
      <li>
        <strong>Site Name:</strong> Enter the name for your website (e.g., "My Awesome Joomla Site").
      </li>
      <li>
        <strong>Site Description:</strong> Add a tagline or brief description for your website.
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    In the **Admin Account** section, configure your Joomla administrator credentials:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a secure username for your Joomla administrator account. Avoid common usernames like "admin" or "administrator".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and unique password.
      </li>
      <li>
        <strong>Real Name:</strong> Enter the real name associated with the admin account.
      </li>
      <li>
        <strong>Admin Email:</strong> Provide a valid email address for administrative purposes.
      </li>
    </ul>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as a text file, as you will need them to log in to your Joomla admin area later.
    </Note>
    <Frame>
      <img
        src="/images/kb/softaculous-install-screen2.gif"
        alt="Joomla admin account setup in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Language">
    Select the preferred language for your Joomla installation from the dropdown menu. English is the default.
    <Frame>
      <img
        src="/images/kb/softaculous-choose-language.gif"
        alt="Joomla language selection in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database settings or set up automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the Joomla installation.
    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Final install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

Once the installation is complete, you will be provided with links to your new Joomla website and its administrative area.
