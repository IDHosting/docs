---
title: "What is Softaculous?"
description: "Learn about Softaculous, an auto-installer that simplifies the process of installing and managing web applications."
---

Softaculous is an auto-installer that allows for the automated installation of scripts with a single click. It supports a wide variety of popular scripts and enables automatic upgrades without any hassle.

You can install applications like WordPress, Joomla, Drupal, MyBB Forum, SMF, Mambo, and many others through the Softaculous Auto-Script Installer.

With Softaculous, you can also perform actions such as:
*   Automatically install scripts
*   Take backups of your scripts
*   Restore backups
*   Uninstall software
*   Utilize the automatic backup creation feature

All these actions can be performed with a single click.
