---
title: "Install FluxBB Forum via Softaculous in SiteWorx"
description: "A guide on how to install FluxBB Forum using the Softaculous installer in SiteWorx."
---

Softaculous is a one-click installer that simplifies the process of adding web applications to your website. This guide will walk you through installing FluxBB Forum using Softaculous within your SiteWorx control panel.

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Begin by logging into your SiteWorx account.
  </Step>
  <Step title="Access Softaculous">
    Once logged in, navigate to the **Additional Features** section and click on **Softaculous**.

    <Frame>
      <img 
        src="/images/kb/sw-softaculous-icon.gif" 
        alt="SiteWorx Softaculous Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Navigate to FluxBB in Softaculous">
    SiteWorx will redirect you to the Softaculous interface. On the left-hand sidebar, type **FluxBB** into the search box. Click on the **FluxBB** link that appears in the search results.

    <Frame>
      <img 
        src="/images/kb/fluxbb-softaculous.gif" 
        alt="Softaculous FluxBB Search Result"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the FluxBB installation process.

    <Frame>
      <img 
        src="/images/kb/fluxbb-install-button.gif" 
        alt="FluxBB Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Under the **Software Setup** section, configure the following:
    <ul>
      <li><strong>Choose Installation URL:</strong> Select the desired protocol (HTTP or HTTPS) and domain. In the "In Directory" field, enter a directory name if you wish to install FluxBB in a subdirectory (e.g., `example.com/forum`). Leave this field blank to install FluxBB on your main domain (e.g., `example.com`).</li>
    </ul>

    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous General Software Setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide the following details:
    <ul>
      <li><strong>Site Name:</strong> Enter a name for your FluxBB forum (e.g., "Community Forum").</li>
      <li><strong>Site Description:</strong> Provide a brief description of your forum.</li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    In the **Admin Account** section, enter:
    <ul>
      <li><strong>Admin Username:</strong> Choose a username for your forum's administrator. For security reasons, avoid common usernames like "admin" or "administrator".</li>
      <li><strong>Admin Password:</strong> Create a strong and secure password.</li>
      <li><strong>Admin Email:</strong> Enter your email address. This is crucial for password recovery and notifications.</li>
    </ul>

    <Note>
      It is highly recommended to copy and save your admin username and password to a secure location, such as a Notepad file. You will need these credentials to access the FluxBB admin area later.
    </Note>

    <Frame>
      <img 
        src="/images/kb/softaculous-screen2-general.gif" 
        alt="Softaculous Admin Account Setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select Language">
    Choose the desired language for your FluxBB forum from the **Choose Language** dropdown. The default language is English.

    <Frame>
      <img 
        src="/images/kb/softaculous-choose-language.gif" 
        alt="Softaculous Choose Language"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows for more granular control, such as changing the database name or configuring automated backups. For a standard installation, you can typically leave these settings at their defaults. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the FluxBB forum installation.

    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous Final Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
