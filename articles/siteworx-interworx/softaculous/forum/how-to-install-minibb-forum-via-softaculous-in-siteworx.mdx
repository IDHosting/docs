---
title: "Install miniBB Forum via Softaculous in SiteWorx"
description: "A guide on how to install miniBB Forum using the Softaculous installer in SiteWorx."
---

Softaculous is a one-click installer that simplifies the process of adding web applications to your website. This guide will walk you through installing miniBB Forum using Softaculous within your SiteWorx control panel.

## Installation Steps

<Steps>
  <Step title="Log in to SiteWorx">
    Begin by logging into your SiteWorx account.
  </Step>
  <Step title="Access Softaculous">
    Navigate to the **Additional Features** section and click on **Softaculous**.
    <Frame>
      <img 
        src="/images/kb/sw-softaculous-icon.gif" 
        alt="Softaculous icon in SiteWorx"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Navigate Softaculous Interface">
    SiteWorx will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for miniBB">
    On the left-hand side of the Softaculous panel, use the search box to type `miniBB`. Click on the `miniBB` link that appears in the search results.
    <Frame>
      <img 
        src="/images/kb/minibb-softaculous.gif" 
        alt="Searching for miniBB in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img 
        src="/images/kb/minibb-install-button.gif" 
        alt="Install button for miniBB"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Under the **Software Setup** section, configure the following:
    *   **Choose Installation URL:** Select the desired protocol (HTTP or HTTPS) and your domain.
    *   **In Directory:** If you wish to install miniBB in a subdirectory (e.g., `example.com/forum`), enter the directory name here. Leave this field blank to install miniBB on your main domain (e.g., `example.com`).
    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Software setup screen in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Admin Account">
    In the **Admin Account** section, provide the following details:
    *   **Admin Username:** Choose a username for your miniBB administrator. For security, avoid common usernames like `admin` or `administrator`.
    *   **Admin Password:** Create a strong and secure password.
    *   **Admin Email:** Enter your email address. This is crucial for password recovery and important notifications.
    <Frame>
      <img 
        src="/images/kb/softaculous-general-admin-fields.gif" 
        alt="Admin account fields in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>It is highly recommended to copy your admin username and password to a secure location, such as a Notepad file. You will need these credentials to access the miniBB admin control panel later.</Note>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows for further customization, such as changing the database name or configuring automated backups. Be mindful that frequent backups can consume significant disk space. For a standard installation, you can leave these settings at their defaults.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation process.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Final install button in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
