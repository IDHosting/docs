---
title: "Install Advanced Electron Forum (AEF) via Softaculous in SiteWorx"
description: "A step-by-step guide on how to install Advanced Electron Forum (AEF) using Softaculous in your SiteWorx account."
---

<Info>Softaculous simplifies the installation of web applications. This guide will walk you through installing Advanced Electron Forum (AEF).</Info>

<Steps>
  <Step title="Log in to your SiteWorx account">
    Access your SiteWorx control panel.
  </Step>
  <Step title="Navigate to Softaculous">
    In the **Additional Features** section, click on **Softaculous**.
    <Frame>
      <img 
        src="/images/kb/sw-softaculous-icon.gif" 
        alt="SiteWorx Softaculous Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    SiteWorx will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for AEF">
    On the left-hand side, type `AEF` into the search box. Click on the **AEF** link that appears.
    <Frame>
      <img 
        src="/images/kb/aef-softaculous.gif" 
        alt="Softaculous search for AEF"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click on the **Install** button.
    <Frame>
      <img 
        src="/images/kb/aef-install-button.gif" 
        alt="AEF Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Under the **Software Setup** section, fill in the necessary details:
    <ul>
      <li><strong>Choose Installation URL:</strong> Select the appropriate protocol and domain. In the "<strong>In Directory</strong>" field, enter a directory name if you wish to install AEF in a subdirectory (e.g., `example.com/directory`). Leave it blank to install in your main domain (e.g., `example.com`).</li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous General Install Screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Board Settings">
    Under the **Board Settings** section, enter the following information:
    <ul>
      <li><strong>Board Name:</strong> The name of your forum.</li>
      <li><strong>Board E-mail:</strong> The email address for the forum.</li>
      <li><strong>Use UTF-8 Character Set:</strong> Select this option if you plan to use multiple languages.</li>
    </ul>
  </Step>
  <Step title="Configure Admin Account">
    Under the **Admin Account** section, provide the following details:
    <ul>
      <li><strong>Admin Username:</strong> Your new AEF username. For security, avoid using `administrator` or `admin`.</li>
      <li><strong>Admin Password:</strong> Choose a strong password.</li>
      <li><strong>Admin Email:</strong> Your email address, which is useful for password resets and notifications.</li>
    </ul>
    <Note>Copy your admin username and password to a safe place, like Notepad. You will need these to access your AEF forum's admin area later.</Note>
    <Frame>
      <img 
        src="/images/kb/aef-install-screen.gif" 
        alt="AEF Install Screen Configuration"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to change the database name or configure automated backups. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click on **Install**.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
