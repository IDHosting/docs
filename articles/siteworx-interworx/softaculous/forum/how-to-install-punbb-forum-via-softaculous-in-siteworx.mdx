---
title: "Install PunBB Forum via Softaculous in SiteWorx"
description: "A guide on how to install PunBB Forum using the Softaculous installer in your SiteWorx account."
---

Softaculous simplifies the installation of web applications. Follow these steps to install PunBB Forum.

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Begin by logging into your SiteWorx account.
  </Step>
  <Step title="Navigate to Softaculous">
    In the **Additional Features** section, click on **Softaculous**.
    <Frame>
      <img
        src="/images/kb/sw-softaculous-icon.gif"
        alt="SiteWorx Softaculous Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Softaculous Interface">
    SiteWorx will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for PunBB">
    On the left-hand side, type **PunBB** into the search box. Then, click on the **PunBB** link that appears.
    <Frame>
      <img
        src="/images/kb/punbb-softaculous.gif"
        alt="Softaculous PunBB Search Result"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click on the **Install** button.
    <Frame>
      <img
        src="/images/kb/punbb-install-button.gif"
        alt="PunBB Softaculous Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the appropriate protocol (e.g., http, https) and domain. In the "In Directory" field, enter the desired directory name if you wish to install PunBB in a subdirectory (e.g., `example.com/forum`). Leave this field blank to install in the root of your domain (e.g., `example.com`).
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Softaculous General Software Setup Screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Enter the following details under the **Site Settings** section:
    <ul>
      <li>
        <strong>Site Name:</strong> Provide a name for your forum (e.g., "Technology Forum").
      </li>
      <li>
        <strong>Site Description:</strong> Enter a brief description for your site.
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, provide the following information:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a username for your PunBB administrator account. For security reasons, avoid common usernames like "administrator" or "admin".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and secure password.
      </li>
      <li>
        <strong>Admin Email:</strong> Enter your email address. This is crucial for password resets and notifications.
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/softaculous-screen2-general.gif"
        alt="Softaculous Admin Account Setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as a Notepad file. You will need these credentials to access the admin area of your PunBB forum later.
    </Note>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows for further customization, such as changing the database name or configuring automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll down to the bottom of the page and click the **Install** button to finalize the installation process.
    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous Final Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
