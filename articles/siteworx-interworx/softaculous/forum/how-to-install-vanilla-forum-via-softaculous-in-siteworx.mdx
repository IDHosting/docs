---
title: "Install Vanilla Forum via Softaculous in SiteWorx"
description: "A guide on how to install Vanilla Forum using the Softaculous installer in SiteWorx."
---

Softaculous is a one-click installer that simplifies the process of adding web applications to your website. This guide will walk you through installing Vanilla Forum using Softaculous within your SiteWorx control panel.

<Steps>
  <Step title="Log in to your SiteWorx account">
    Begin by accessing your SiteWorx account.
  </Step>
  <Step title="Navigate to Softaculous">
    Once logged in, locate the **Additional Features** section and click on **Softaculous**.

    <Frame>
      <img
        src="/images/kb/sw-softaculous-icon.gif"
        alt="Softaculous icon in SiteWorx"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    SiteWorx will redirect you to the Softaculous application installer.
  </Step>
  <Step title="Search for Vanilla Forum">
    On the left-hand side of the Softaculous interface, use the search box to type "Vanilla". Click on the **Vanilla** link that appears in the search results.

    <Frame>
      <img
        src="/images/kb/vanilla-softaculous.gif"
        alt="Searching for Vanilla in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the Vanilla Forum installation process.

    <Frame>
      <img
        src="/images/kb/vanilla-install-button.gif"
        alt="Vanilla install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Under the **Software Setup** section, configure the following:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the desired protocol (HTTP or HTTPS) and your domain. In the <strong>In Directory</strong> field, enter a directory name if you wish to install Vanilla Forum in a subdirectory (e.g., `example.com/forum`). Leave this field blank to install it on your main domain (e.g., `example.com`).
      </li>
    </ul>

    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Softaculous general install screen - Software Setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    In the **Site Settings** section, provide the following:
    <ul>
      <li>
        <strong>Site Name:</strong> Enter a name for your forum, such as "Community Hub" or "Tech Discussions".
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter your administrator credentials:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a username for your Vanilla forum administrator. For security, avoid common usernames like "admin" or "administrator".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and secure password.
      </li>
      <li>
        <strong>Admin Email:</strong> Provide a valid email address. This is crucial for password recovery and important notifications.
      </li>
    </ul>

    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as Notepad, as you will need them to access your Vanilla forum's administrative area later.
    </Note>

    <Frame>
      <img
        src="/images/kb/softaculous-gen-install-screen-two.gif"
        alt="Softaculous general install screen - Admin Account"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows for further customization, such as changing the database name or configuring automated backups. Unless you have specific requirements, you can leave these settings at their defaults. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation of Vanilla Forum.

    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
