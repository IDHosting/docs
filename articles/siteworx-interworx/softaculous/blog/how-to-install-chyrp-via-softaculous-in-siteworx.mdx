---
title: "Install Chyrp via Softaculous in SiteWorx"
description: "A guide on how to install Chyrp using the Softaculous installer within your SiteWorx control panel."
---

Softaculous is a one-click installer that simplifies the process of adding web applications to your website. This guide will walk you through installing Chyrp using Softaculous in your SiteWorx account.

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Begin by logging into your SiteWorx control panel.
  </Step>
  <Step title="Navigate to Softaculous">
    Once logged in, locate the **Additional Features** section and click on **Softaculous**.

    <Frame>
      <img 
        src="/images/kb/sw-softaculous-icon.gif" 
        alt="SiteWorx Softaculous Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    SiteWorx will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for Chyrp">
    On the left-hand side of the Softaculous panel, use the search box to type "Chyrp". Click on the **Chyrp** link that appears in the search results.

    <Frame>
      <img 
        src="/images/kb/chyrp-softaculous.gif" 
        alt="Softaculous Chyrp Search Result"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the Chyrp installation process.

    <Frame>
      <img 
        src="/images/kb/chyrp-install-button.gif" 
        alt="Chyrp Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Under the **Software Setup** section, configure the following:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the desired protocol (e.g., http, https) and domain. In the <strong>In Directory</strong> field, enter a directory name if you wish to install Chyrp in a subdirectory (e.g., `example.com/blog`). Leave this field blank to install Chyrp in the root of your domain (e.g., `example.com`).
      </li>
    </ul>

    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous Software Setup Screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    In the **Site Settings** section, provide the following details:
    <ul>
      <li>
        <strong>Site Name:</strong> Enter a name for your Chyrp website (e.g., "My Awesome Blog").
      </li>
      <li>
        <strong>Site Description:</strong> Add a brief description of your website.
      </li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, configure your administrator credentials:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a username for your Chyrp admin account. For security, avoid common usernames like "admin" or "administrator".
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and secure password.
      </li>
      <li>
        <strong>Admin Email:</strong> Enter your email address.
      </li>
    </ul>

    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as a text file, as you will need them to log in to your Chyrp admin area later.
    </Note>

    <Frame>
      <img 
        src="/images/kb/softaculous-screen2-general.gif" 
        alt="Softaculous Admin Account Setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database settings and automated backups. You can typically leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation of Chyrp.

    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous Final Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
