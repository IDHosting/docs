---
title: "Install Serendipity via Softaculous in SiteWorx"
description: "A guide on how to install Serendipity using the Softaculous installer in SiteWorx."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. This guide will walk you through the process of installing Serendipity using Softaculous within your SiteWorx account.

<Steps>
  <Step title="Log in to SiteWorx">
    Begin by logging into your SiteWorx account.
  </Step>
  <Step title="Access Softaculous">
    Navigate to the **Additional Features** section and click on **Softaculous**.

    <Frame>
      <img 
        src="/images/kb/sw-softaculous-icon.gif" 
        alt="SiteWorx Softaculous Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Navigate to Serendipity">
    You will be redirected to the Softaculous interface. On the left-hand side, type **Serendipity** into the search box. Click on the **Serendipity** link that appears.

    <Frame>
      <img 
        src="/images/kb/serendipity-softaculous.gif" 
        alt="Softaculous Serendipity Search"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.

    <Frame>
      <img 
        src="/images/kb/serendipity-install-button.gif" 
        alt="Serendipity Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Under the **Software Setup** section, configure the following:
    *   **Choose Installation URL**: Select the appropriate protocol (HTTP or HTTPS) and your domain.
    *   **In Directory**: If you wish to install Serendipity in a subdirectory (e.g., `example.com/blog`), enter the directory name here. Leave it blank to install in the root of your domain (e.g., `example.com`).

    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous General Install Screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    In the **Site Settings** section, provide:
    *   **Site Name**: The name of your Serendipity blog (e.g., "My Personal Blog").
    *   **Site Description**: A brief description of your blog.
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter:
    *   **Admin Username**: Your desired username for the Serendipity admin area. Avoid common usernames like "admin" for security.
    *   **Admin Password**: Create a strong and secure password.
    *   **Real Name**: Your full name.
    *   **Admin Email**: Your email address.

    <Note>
      It is highly recommended to copy your Admin Username and Password to a secure location, such as a password manager or a text file, as you will need them to log in to your Serendipity admin area later.
    </Note>

    <Frame>
      <img 
        src="/images/kb/softaculous-install-screen2.gif" 
        alt="Softaculous Install Screen - Admin Account"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Language">
    Select the desired language for your Serendipity blog from the **Choose Language** dropdown. The default is English.

    <Frame>
      <img 
        src="/images/kb/softaculous-choose-language.gif" 
        alt="Softaculous Choose Language"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database settings and automated backups. You can typically leave these at their default values unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation.

    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous Final Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

Once the installation is complete, you will be provided with links to your new Serendipity blog and its administration area.
