---
title: "Install Nucleus via Softaculous in SiteWorx"
description: "A guide on how to install Nucleus using the Softaculous installer within your SiteWorx account."
---

Softaculous simplifies the process of installing web applications. Follow these steps to install Nucleus on your SiteWorx account.

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Access your SiteWorx control panel.
  </Step>
  <Step title="Navigate to Softaculous">
    In the **Additional Features** section, click on **Softaculous**.
    <Frame>
      <img 
        src="/images/kb/sw-softaculous-icon.gif" 
        alt="SiteWorx Softaculous Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Search for Nucleus">
    SiteWorx will redirect you to the Softaculous interface. On the left-hand side, type **Nucleus** into the search box. Click the **Nucleus** link that appears.
    <Frame>
      <img 
        src="/images/kb/nucleus-softaculous.gif" 
        alt="Softaculous search for Nucleus"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img 
        src="/images/kb/nucleus-install-button.gif" 
        alt="Nucleus Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL:</strong> Select the appropriate protocol and domain. In the <strong>In Directory</strong> field, enter a directory name if you wish to install Nucleus in a subdirectory (e.g., `example.com/directory`). Leave it blank to install in your main domain (e.g., `example.com`).</li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-general-install-screen.gif" 
        alt="Softaculous General Install Screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the <strong>Site Settings</strong> section, enter the following:
    <ul>
      <li><strong>Site Name:</strong> The name of your website (e.g., <strong>My First Website</strong>).</li>
      <li><strong>Site Short Name:</strong> A short name for your site.</li>
    </ul>
  </Step>
  <Step title="Configure Admin Account">
    Under the <strong>Admin Account</strong> section, provide the following details:
    <ul>
      <li><strong>Admin Username:</strong> Your new Nucleus username. For security, avoid using generic usernames like "administrator" or "admin".</li>
      <li><strong>Admin Password:</strong> Create a strong password.</li>
      <li><strong>Real Name:</strong> Your real name or leave as is.</li>
      <li><strong>Admin Email:</strong> Your email address.</li>
    </ul>
    <Note>
      Copy your admin username and password to a safe place, such as Notepad. You will need these credentials to access the Nucleus admin area later.
    </Note>
    <Frame>
      <img 
        src="/images/kb/mycms-install-screen.gif" 
        alt="Nucleus Install Screen - Admin Account"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options and Finalize">
    Under <strong>Advanced Options</strong>, you can configure database names or set up automated backups. Be mindful that frequent backups can consume significant disk space.
    <br />
    <br />
    Once you have configured all settings, scroll to the bottom of the page and click the **Install** button.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
