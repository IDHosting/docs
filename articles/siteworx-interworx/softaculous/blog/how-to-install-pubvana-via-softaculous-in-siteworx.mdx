---
title: "Install Pubvana via Softaculous in SiteWorx"
description: "A guide on how to install Pubvana using the Softaculous installer within your SiteWorx account."
---

Softaculous is a fantastic tool that simplifies the installation of web applications. This guide will walk you through the steps to install Pubvana using Softaculous in your SiteWorx control panel.

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Begin by logging into your SiteWorx account.
  </Step>
  <Step title="Navigate to Softaculous">
    Once logged in, locate the **Additional Features** section in your SiteWorx dashboard. Click on **Softaculous**.

    <Frame>
      <img 
        src="/images/kb/sw-softaculous-icon.gif" 
        alt="SiteWorx Softaculous Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Softaculous Interface">
    SiteWorx will automatically redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for Pubvana">
    On the left-hand side of the Softaculous interface, you'll find a search box. Type **Pubvana** into this box. The **Pubvana** link will appear below the search bar; click on it.

    <Frame>
      <img 
        src="/images/kb/pubvana-softaculous.gif" 
        alt="Softaculous search for Pubvana"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    After selecting Pubvana, click the **Install** button.

    <Frame>
      <img 
        src="/images/kb/pubvana-install-button.gif" 
        alt="Pubvana Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Under the **Software Setup** section, you will need to configure the following:
    *   **Choose Installation URL**: Select the appropriate protocol (HTTP or HTTPS) and your domain. If you wish to install Pubvana in a subdirectory (e.g., `example.com/directory`), enter the directory name in the **In Directory** field. Leave this field blank to install Pubvana in the root of your domain (e.g., `example.com`).

        <Frame>
          <img 
            src="/images/kb/softaculous-general-install-screen.gif" 
            alt="Softaculous General Install Screen"
            style={{ width: '100%', height: 'auto' }}
          />
        </Frame>
  </Step>
  <Step title="Configure Site Settings">
    In the **Site Settings** section, provide the following details:
    *   **Site Name**: Enter a name for your website (e.g., "My First Pubvana Blog").
    *   **Site Description**: Add a brief description of your website.
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter the following credentials:
    *   **Admin Email**: Your email address.
    *   **Admin Password**: Choose a strong and secure password.
    *   **Admin Username**: Create a username for your Pubvana administrator. For security, avoid using generic names like "admin" or "administrator".
    *   **First Name**: Your first name.
    *   **Last Name**: Your last name.

    <Frame>
      <img 
        src="/images/kb/pubvana-screen2.gif" 
        alt="Pubvana Admin Account Setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>

    <Note>
      It is highly recommended to copy and save your admin username and password to a secure location, such as a Notepad file. You will need these credentials to log in to your Pubvana admin area later.
    </Note>
  </Step>
  <Step title="Advanced Options and Finalize Installation">
    The **Advanced Options** section allows you to configure settings like the database name or set up automated backups. You can generally leave these as default unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.

    Once you have configured all the necessary fields, scroll to the bottom of the page and click the **Install** button.

    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous Install Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

Pubvana should now be successfully installed on your domain via Softaculous. You can access your new Pubvana installation using the details you provided.
