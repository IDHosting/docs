---
title: "Install WordPress via Softaculous in SiteWorx"
description: "A guide on how to install WordPress using the Softaculous installer within your SiteWorx account."
---

Softaculous is a one-click installer that simplifies the process of installing web applications. This guide will walk you through installing WordPress using Softaculous in your SiteWorx account.

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Access your SiteWorx control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to Softaculous">
    In the **Additional Features** section, click on the **Softaculous** icon.
    <Frame>
      <img 
        src="/images/kb/sw-softaculous-icon.gif" 
        alt="Softaculous icon in SiteWorx"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Softaculous Interface">
    SiteWorx will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for WordPress">
    On the left-hand side of the Softaculous panel, type "WordPress" into the search box. Click on the **WordPress** link that appears in the search results.
    <Frame>
      <img 
        src="/images/kb/wordpress-softaculous.gif" 
        alt="Searching for WordPress in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate WordPress Installation">
    Click the **Install** button to begin the WordPress installation process.
    <Frame>
      <img 
        src="/images/kb/wordpress-install-button.gif" 
        alt="WordPress install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL</strong>: Select the desired protocol (e.g., http, https) and your domain. If you wish to install WordPress in a subdirectory (e.g., `example.com/directory`), enter the directory name in the "In Directory" field. Leave it blank to install in your main domain (e.g., `example.com`).</li>
      <li><strong>Choose the version you want to install:</strong> You can typically leave this at the default version.</li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/softaculous-default-install-screen.gif" 
        alt="Softaculous default installation screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, provide the following information:
    <ul>
      <li><strong>Site Name:</strong> Enter the name of your website (e.g., "My Awesome Blog").</li>
      <li><strong>Site Description:</strong> A brief description or tagline for your website.</li>
      <li><strong>Enable Multisite (WPMU):</strong> Leave this unchecked unless you specifically need a WordPress Multisite setup.</li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, enter:
    <ul>
      <li><strong>Admin Username:</strong> Choose a secure username for your WordPress administrator account. Avoid common usernames like "admin".</li>
      <li><strong>Admin Password:</strong> Create a strong and complex password.</li>
      <li><strong>Admin Email:</strong> Provide a valid email address for administrative purposes.</li>
    </ul>
    <Note>It is highly recommended to copy your Admin Username and Password to a secure location, such as Notepad, for future reference. You will need these to log in to your WordPress admin area.</Note>
    <Frame>
      <img 
        src="/images/kb/wp-install-screen2.gif" 
        alt="WordPress admin account setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select Language">
    Choose the primary language for your WordPress installation from the dropdown menu. The default is English.
    <Frame>
      <img 
        src="/images/kb/softaculous-select-language.gif" 
        alt="Softaculous language selection"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select Plugins (Optional)">
    You can choose to install additional plugins at this stage or skip this step.
  </Step>
  <Step title="Advanced Options (Optional)">
    This section allows you to configure advanced settings such as database name or automated backups. It's generally recommended to leave these as default unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Select Theme (Optional)">
    You have the option to select a WordPress theme during installation. If you prefer to choose a theme later, you can leave this setting as is, and WordPress will use its default theme.
  </Step>
  <Step title="Complete Installation">
    Scroll down to the bottom of the page and click the **Install** button to finalize the WordPress installation.
    <Frame>
      <img 
        src="/images/kb/softaculous-install-button.gif" 
        alt="Softaculous final install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

Once the installation is complete, you will be provided with the URLs to access your new WordPress website and its administration area.
