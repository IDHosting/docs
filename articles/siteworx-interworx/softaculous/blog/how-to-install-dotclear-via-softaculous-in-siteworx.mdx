---
title: "Install Dotclear via Softaculous in SiteWorx"
description: "A guide on how to install Dotclear using the Softaculous installer within your SiteWorx control panel."
---

Softaculous is a one-click installer that simplifies the process of installing web applications. Follow these steps to install Dotclear on your SiteWorx account.

<Steps>
  <Step title="Log in to SiteWorx">
    Begin by logging into your SiteWorx account.
  </Step>
  <Step title="Access Softaculous">
    Navigate to the **Additional Features** section and click on **Softaculous**.
    <Frame>
      <img
        src="/images/kb/sw-softaculous-icon.gif"
        alt="Softaculous icon in SiteWorx"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Navigate to Dotclear">
    SiteWorx will redirect you to the Softaculous interface. On the left-hand side, type **Dotclear** into the search box. Click on the **Dotclear** link that appears.
    <Frame>
      <img
        src="/images/kb/dotclear-softaculous.gif"
        alt="Dotclear search result in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img
        src="/images/kb/dotclear-install-button.gif"
        alt="Dotclear install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    *   **Choose Installation URL**: Select the appropriate protocol (HTTP or HTTPS) and domain. In the **In Directory** field, enter a directory name if you wish to install Dotclear in a subdirectory (e.g., `example.com/directory`). Leave this field blank to install Dotclear in the root of your domain (e.g., `example.com`).
    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Softaculous general installation screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, enter the following:
    *   **Site Name**: Provide a name for your Dotclear website (e.g., "My First Website").
  </Step>
  <Step title="Configure Admin Account">
    Under the **Admin Account** section, enter your administrator details:
    *   **Admin Username**: Choose a username for your Dotclear administrator account. For security, avoid common usernames like "admin" or "administrator".
    *   **Admin Password**: Create a strong password for your admin account.
    *   **First Name**: Enter your first name.
    *   **Last Name**: Enter your last name.
    *   **Admin Email**: Provide your email address.
    <Frame>
      <img
        src="/images/kb/softaculous-gen-install-screen.gif"
        alt="Softaculous admin account setup"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location, such as a text file, as you will need them to log in to your Dotclear admin area later.
    </Note>
  </Step>
  <Step title="Choose Language">
    Select the desired language for your Dotclear blog from the **Choose Language** dropdown. The default language is English.
    <Frame>
      <img
        src="/images/kb/softaculous-choose-language.gif"
        alt="Softaculous language selection"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options and Install">
    You can review the **Advanced Options** if you wish to change the database name or configure automated backups. Be mindful that frequent backups can consume significant disk space.
    <br />
    Finally, scroll to the bottom of the page and click the **Install** button.
    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
