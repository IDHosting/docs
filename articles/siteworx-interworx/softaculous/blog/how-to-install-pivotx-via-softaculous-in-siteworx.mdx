---
title: "Install PivotX via Softaculous in SiteWorx"
description: "A guide on how to install PivotX using the Softaculous installer within your SiteWorx account."
---

Softaculous simplifies the installation of web applications. Follow these steps to install PivotX on your SiteWorx hosting account.

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Access your SiteWorx control panel.
  </Step>
  <Step title="Navigate to Softaculous">
    In the **Additional Features** section, click on **Softaculous**.
    <Frame>
      <img
        src="/images/kb/sw-softaculous-icon.gif"
        alt="Softaculous icon in SiteWorx"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Softaculous Interface">
    SiteWorx will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for PivotX">
    On the left-hand side of the Softaculous panel, type `PivotX` into the search box. Click on the **PivotX** link that appears in the search results.
    <Frame>
      <img
        src="/images/kb/pivotx-softaculous.gif"
        alt="PivotX search result in Softaculous"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img
        src="/images/kb/pivotx-install-button.gif"
        alt="PivotX install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li>
        <strong>Choose Installation URL:</strong> Select the desired protocol (HTTP or HTTPS) and domain. In the <strong>In Directory</strong> field, enter a directory name if you wish to install PivotX in a subdirectory (e.g., `example.com/directory`). Leave this field blank to install PivotX in the root of your domain (e.g., `example.com`).
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/softaculous-general-install-screen.gif"
        alt="Softaculous general installation screen"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, enter the following:
    <ul>
      <li>
        <strong>Site Name:</strong> Provide a name for your website (e.g., `My First Website`).
      </li>
    </ul>
  </Step>
  <Step title="Configure Admin Account">
    Under the **Admin Account** section, enter the following details:
    <ul>
      <li>
        <strong>Admin Username:</strong> Choose a username for your PivotX administrator account. For security, avoid using common usernames like `admin` or `administrator`.
      </li>
      <li>
        <strong>Admin Password:</strong> Create a strong and secure password.
      </li>
      <li>
        <strong>Nickname:</strong> Enter a nickname for the admin user.
      </li>
      <li>
        <strong>Admin Email:</strong> Provide your email address.
      </li>
    </ul>
    <Frame>
      <img
        src="/images/kb/softaculous-install-screen-nickname.gif"
        alt="Softaculous admin account configuration"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      It is highly recommended to copy your admin username and password to a secure location (like Notepad) immediately after setting them. You will need these credentials to log in to your PivotX blog's admin area later.
    </Note>
  </Step>
  <Step title="Choose Language">
    Select the desired language for your PivotX blog from the dropdown menu. The default language is English.
    <Frame>
      <img
        src="/images/kb/softaculous-choose-language.gif"
        alt="Softaculous language selection"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows for more specific configurations, such as changing the database name or setting up automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the PivotX installation.
    <Frame>
      <img
        src="/images/kb/softaculous-install-button.gif"
        alt="Softaculous final install button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>
