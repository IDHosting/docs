---
title: "Install b2evolution via Softaculous in SiteWorx"
description: "A guide on how to install b2evolution using the Softaculous installer in your SiteWorx account."
---

Softaculous is a one-click installer that simplifies the process of adding web applications to your website. This guide will walk you through installing b2evolution using Softaculous within your SiteWorx control panel.

## Installation Steps

<Steps>
  <Step title="Log in to your SiteWorx Account">
    Begin by logging into your SiteWorx account.
  </Step>
  <Step title="Navigate to Softaculous">
    Once logged in, locate the **Additional Features** section and click on **Softaculous**.
    <Frame>
      <img src="/images/kb/sw-softaculous-icon.gif" alt="SiteWorx Softaculous Icon" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Access Softaculous Interface">
    SiteWorx will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for b2evolution">
    On the left-hand side of the Softaculous interface, use the search box to type `b2evolution`. Click on the `b2evolution` link that appears in the search results.
    <Frame>
      <img src="/images/kb/b2evolution-softaculous.gif" alt="Softaculous search for b2evolution" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button to begin the b2evolution installation process.
    <Frame>
      <img src="/images/kb/b2evolution-install-button.gif" alt="b2evolution install button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL</strong>: Select the desired protocol (HTTP or HTTPS) and domain. In the <strong>In Directory</strong> field, enter a directory name if you wish to install b2evolution in a subdirectory (e.g., `example.com/blog`). Leave this field blank to install it in the root of your domain (e.g., `example.com`).</li>
    </ul>
    <Frame>
      <img src="/images/kb/softaculous-general-install-screen.gif" alt="Softaculous general install screen" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Set Up Admin Account">
    Under the <strong>Admin Account</strong> section, provide the following details:
    <ul>
      <li><strong>Admin Username:</strong> Create a username for your b2evolution administrator. For security, avoid using common usernames like "admin" or "administrator".</li>
      <li><strong>Admin Password:</strong> Choose a strong and secure password.</li>
      <li><strong>Admin Email:</strong> Enter your email address.</li>
    </ul>
    <Frame>
      <img src="/images/kb/softaculous-general-admin-fields.gif" alt="Softaculous general admin fields" style={{ width: '100%', height: 'auto' }} />
    </Frame>
    <Note>It is highly recommended to copy your admin username and password to a secure location, such as a Notepad file. You will need these credentials to log in to your b2evolution blog's administration area later.</Note>
  </Step>
  <Step title="Choose Language">
    Select the preferred language for your b2evolution blog from the available options. The default language is English.
    <Frame>
      <img src="/images/kb/softaculous-choose-language.gif" alt="Softaculous choose language" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The <strong>Advanced Options</strong> section allows you to configure database settings or set up automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to finalize the installation.
    <Frame>
      <img src="/images/kb/softaculous-install-button.gif" alt="Softaculous install button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
</Steps>
