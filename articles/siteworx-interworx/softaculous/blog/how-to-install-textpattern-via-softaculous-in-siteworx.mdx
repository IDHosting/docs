---
title: "Install Textpattern via Softaculous in SiteWorx"
description: "A guide on how to install Textpattern using the Softaculous installer within your SiteWorx account."
---

Softaculous provides an easy way to install web applications like Textpattern. Follow these steps to install Textpattern on your website.

<Steps>
  <Step title="Log in to SiteWorx">
    Access your SiteWorx account by logging in.
  </Step>
  <Step title="Navigate to Softaculous">
    In the **Additional Features** section, click on **Softaculous**.
    <Frame>
      <img src="/images/kb/sw-softaculous-icon.gif" alt="Softaculous icon in SiteWorx" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Access Softaculous Interface">
    SiteWorx will redirect you to the Softaculous interface.
  </Step>
  <Step title="Search for Textpattern">
    On the left-hand side of the Softaculous panel, type **Textpattern** into the search box. Click on the **Textpattern** link that appears in the search results.
    <Frame>
      <img src="/images/kb/textpattern-softaculous.gif" alt="Textpattern search result in Softaculous" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Initiate Installation">
    Click the **Install** button.
    <Frame>
      <img src="/images/kb/textpattern-install-button.gif" alt="Textpattern install button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Software Setup">
    Fill in the necessary fields under the **Software Setup** section:
    <ul>
      <li><strong>Choose Installation URL:</strong> Select the appropriate protocol (HTTP or HTTPS) and domain. In the "<strong>In Directory</strong>" field, enter a directory name if you wish to install Textpattern in a subdirectory (e.g., `example.com/directory`). Leave this field blank to install in the root of your domain (e.g., `example.com`).</li>
    </ul>
    <Frame>
      <img src="/images/kb/softaculous-general-install-screen.gif" alt="Softaculous general installation screen" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Configure Site Settings">
    Under the **Site Settings** section, enter the following information:
    <ul>
      <li><strong>Site Name:</strong> The name of your website (e.g., <strong>My First Website</strong>).</li>
      <li><strong>Site Description:</strong> A brief description of your site.</li>
    </ul>
  </Step>
  <Step title="Set Up Admin Account">
    Under the **Admin Account** section, provide the following details:
    <ul>
      <li><strong>Admin Username:</strong> Your new Textpattern username. For security, avoid using common usernames like "admin" or "administrator".</li>
      <li><strong>Admin Password:</strong> Create a strong password.</li>
      <li><strong>Real Name:</strong> Your real name or leave as is.</li>
      <li><strong>Admin Email:</strong> Your email address.</li>
    </ul>
    <Note>
      Copy your admin username and password to a safe place, such as Notepad. You will need these credentials to log in to your Textpattern admin area later.
    </Note>
    <Frame>
      <img src="/images/kb/softaculous-install-screen2.gif" alt="Softaculous admin account setup screen" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
  <Step title="Advanced Options (Optional)">
    The **Advanced Options** section allows you to configure database settings or set up automated backups. You can generally leave these settings at their defaults unless you have specific requirements. Be mindful that frequent backups can consume significant disk space.
  </Step>
  <Step title="Complete Installation">
    Scroll to the bottom of the page and click the **Install** button to begin the installation process.
    <Frame>
      <img src="/images/kb/softaculous-install-button.gif" alt="Softaculous install confirmation button" style={{ width: '100%', height: 'auto' }} />
    </Frame>
  </Step>
</Steps>
