---
title: "Edit File in the SiteWorx File Manager"
description: "Learn how to edit files directly within your SiteWorx account using the built-in File Manager."
---

You can use SiteWorx's inbuilt file manager to edit files on your website.

<Steps>
  <Step title="Log in to your SiteWorx account">
    Access your SiteWorx control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to File Manager">
    In the main SiteWorx dashboard, locate and click on the **File Manager** section, then click on **File Manager** again.
    <Frame>
      <img 
        src="/images/kb/sw-file-manager-icon.gif" 
        alt="SiteWorx File Manager Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Locate your file">
    Browse through the directory structure to find the specific file you wish to edit.
    <Frame>
      <img 
        src="/images/kb/siteworx-filemanager-html.gif" 
        alt="SiteWorx File Manager directory navigation"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select the file">
    Click on the name of the file you want to edit.
    <Frame>
      <img 
        src="/images/kb/siteworx-filemanager-file-edit.gif" 
        alt="SiteWorx File Manager selecting a file"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate editing">
    After selecting the file, click the **Edit** button.
    <Frame>
      <img 
        src="/images/kb/siteworx-filemanager-file-edit-option.gif" 
        alt="SiteWorx File Manager Edit button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Make and save changes">
    Modify the content of your file in the editor. Once you are finished, click the **Save** button to apply your changes.
    <Frame>
      <img 
        src="/images/kb/siteworx-filemanager-edit.gif" 
        alt="SiteWorx File Manager editing interface"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

Your file has now been successfully edited.
