---
title: "Create a Directory or Files in the SiteWorx File Manager"
description: "Learn how to create new directories and files using the SiteWorx File Manager."
---

SiteWorx File Manager is a powerful tool for managing your website's files and directories. This guide will walk you through the process of creating new directories and files.

## Creating a New Directory

Follow these steps to create a new directory in your SiteWorx File Manager:

<Steps>
  <Step title="Log in to SiteWorx">
    Access your SiteWorx account by logging in.
  </Step>
  <Step title="Navigate to File Manager">
    In the SiteWorx dashboard, click on the **File Manager** section, then select **File Manager** again.
    <Frame>
      <img 
        src="/images/kb/sw-file-manager-icon.gif" 
        alt="SiteWorx File Manager Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select Domain (if applicable)">
    If you manage multiple domains, choose the correct one from the drop-down menu and click **Login**.
  </Step>
  <Step title="Navigate to Target Directory">
    Browse to the directory where you wish to create the new folder. For example, to create a folder within the `HTML` directory, click on **HTML**.
    <Frame>
      <img 
        src="/images/kb/siteworx-filemanager-html.gif" 
        alt="SiteWorx File Manager HTML Directory"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate New Item Creation">
    Click on the **New Item** button.
    <Frame>
      <img 
        src="/images/kb/siteworx-filemanager-new-item.gif" 
        alt="SiteWorx File Manager New Item Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Create the Directory">
    Select **Folder** as the item type. Enter the desired name for your new directory in the provided field, and then click **Create Now**.
    <Frame>
      <img 
        src="/images/kb/siteworx-filemanager-folder.gif" 
        alt="SiteWorx File Manager Create Folder"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

## Creating a New File (e.g., myfile.txt)

Follow these steps to create a new file within your SiteWorx File Manager:

<Steps>
  <Step title="Initiate New Item Creation">
    Click on the **New Item** button.
  </Step>
  <Step title="Create the File">
    Select **File** as the item type. Enter the desired name for your new file (e.g., `MyFile.txt`) in the provided field, and then click **Create Now**.
    <Frame>
      <img 
        src="/images/kb/siteworx-filemanager-file.gif" 
        alt="SiteWorx File Manager Create File"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

You have now successfully learned how to create both directories and files using the SiteWorx File Manager.
