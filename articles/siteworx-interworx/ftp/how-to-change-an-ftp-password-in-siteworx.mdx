---
title: "Change an FTP Password in SiteWorx"
description: "Learn how to change the password for a specific FTP account within your SiteWorx control panel."
---

To change the password for a specific FTP account in SiteWorx, follow these steps. Note that changing the main root FTP account password requires changing your overall SiteWorx account password.

<Steps>
  <Step title="Log in to SiteWorx">
    Access your SiteWorx account by logging in with your credentials.
  </Step>
  <Step title="Navigate to FTP Accounts">
    In the **FTP** section, click on **Accounts**.
    <Frame>
      <img
        src="/images/kb/sw-ftp-accounts-icon.gif"
        alt="SiteWorx FTP Accounts Icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Edit FTP Account">
    Click on the **Pencil icon** next to the FTP account you wish to modify.
    <Frame>
      <img
        src="/images/kb/sw-ftp-edit-account.gif"
        alt="SiteWorx Edit FTP Account"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Enter New Password">
    Enter your desired new password in the provided field. Alternatively, you can click the **Magic Wand** icon to generate a strong, random password.
    <Frame>
      <img
        src="/images/kb/sw-change-password.gif"
        alt="SiteWorx Change Password Field"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Save Changes">
    Click the **Save** button to confirm your changes.
  </Step>
</Steps>

You can now access the FTP user account using the newly set password.
