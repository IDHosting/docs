---
title: "Change the PHP Version via CloudLinux Selector in cPanel"
description: "Learn how to change your website's PHP version using the CloudLinux Selector tool within your cPanel account."
---

You can use CloudLinux's Select PHP Version tool to easily change your website's PHP version. CloudLinux provides a selection of different PHP versions to choose from.

<Steps>
  <Step title="Log in to your cPanel account">
    Access your cPanel account using your provided credentials.
  </Step>
  <Step title="Navigate to Select PHP Version">
    In the **Software** section of your cPanel dashboard, click on **Select PHP Version**.
    <Frame>
      <img 
        src="/images/kb/cp-cl-select-php-version-icon.gif" 
        alt="Select PHP Version icon in cPanel"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose your desired PHP version">
    To the right of the **Current PHP version**, click the **drop-down menu** and select the PHP version you wish to use for your website.
    <Frame>
      <img 
        src="/images/kb/cp-cl-current-php-version-option.gif" 
        alt="Dropdown menu to select PHP version"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Apply the changes">
    After selecting your desired PHP version, click the **Apply** button to save your changes.
    <Frame>
      <img 
        src="/images/kb/cl-set-as-php-version.gif" 
        alt="Apply button to set the chosen PHP version"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

You have now successfully changed your website's PHP version using the CloudLinux Selector in cPanel.
