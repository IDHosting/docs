---
title: "Enable or Disable PHP Extensions Using the CloudLinux Selector in cPanel"
description: "Learn how to easily enable or disable PHP extensions in cPanel using the CloudLinux Selector."
---

You can use the CloudLinux Selector to enable or disable available PHP extensions in your cPanel account.

<Steps>
  <Step title="Log in to cPanel">
    Begin by logging in to your cPanel account.
  </Step>
  <Step title="Navigate to Select PHP Version">
    In the **Software** section of your cPanel dashboard, click on **Select PHP Version**.
    <Frame>
      <img
        src="/images/kb/cp-cl-select-php-version-icon.gif"
        alt="cPanel Select PHP Version icon"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access the Extensions Tab">
    Once on the PHP Selector page, click on the **Extensions** tab.
    <Frame>
      <img
        src="/images/kb/cl-extensions-tab-button.gif"
        alt="CloudLinux Extensions tab button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      If the **Extensions** page displays an error related to the native PHP version, you will need to change it to a non-native version first before proceeding.
    </Note>
  </Step>
  <Step title="Enable or Disable Extensions">
    To **enable** a PHP extension, check the box next to its name. To **disable** an extension, uncheck the box.
    <Frame>
      <img
        src="/images/kb/cl-enable-disabled-extension.gif"
        alt="CloudLinux enable/disable extension interface"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    The interface will indicate whether the module is currently enabled or disabled.
  </Step>
</Steps>
