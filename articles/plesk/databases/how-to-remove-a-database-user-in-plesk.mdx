---
title: "Remove a Database User in Plesk"
description: "Learn how to remove a database user in Plesk with this step-by-step guide."
---

In this tutorial, you will learn how to remove a database user in Plesk.

<Steps>
  <Step title="Log in to your Plesk account">
    Access your Plesk control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to Databases">
    From the **Sidebar Menu**, click on **Databases**.
    <Frame>
      <img 
        src="/images/kb/plesk-database-menu.gif" 
        alt="Plesk Databases Menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Go to User Management">
    Click on the **User Management** tab.
    <Frame>
      <img 
        src="/images/kb/plesk-database-user-management-menu.gif" 
        alt="Plesk Database User Management Menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select and Remove User">
    Under the **Database Users** section, check the box next to the database user you wish to remove. Then, click the **Remove** button.
    <Frame>
      <img 
        src="/images/kb/plesk-remove-database-user.gif" 
        alt="Plesk Remove Database User"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Confirm Removal">
    A confirmation dialog will appear. Click **Yes** to confirm the removal of the user.
  </Step>
</Steps>

The database user has now been successfully removed in Plesk.
