---
title: "Create a Database in Plesk"
description: "A step-by-step guide on how to create a new database within your Plesk control panel."
---

This article will guide you through the process of creating a new database in Plesk.

<Steps>
  <Step title="Log in to Plesk">
    Begin by logging into your Plesk account.
  </Step>
  <Step title="Navigate to Databases">
    From the **Sidebar Menu**, click on **Databases**.
    <Frame>
      <img
        src="/images/kb/plesk-database-menu.gif"
        alt="Plesk Databases menu item in the sidebar"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Add a New Database">
    Click on the **Add Database** button.
    <Frame>
      <img
        src="/images/kb/plesk-add-database-button.gif"
        alt="Add Database button in Plesk"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Fill in Database Details">
    Under the **Add a Database** section, you will need to provide the following information:
    <ul>
      <li><strong>Database Name:</strong> Enter a name for your database (e.g., <code>blog</code>).</li>
      <li><strong>Related Site:</strong> Select the website from the dropdown menu that this database will be associated with.</li>
      <li><strong>Users:</strong> Uncheck the box for <strong>Create a database user</strong>. You can manage users separately if needed.</li>
    </ul>
    <Frame>
      <img
        src="/images/kb/plesk-add-a-database-final.gif"
        alt="Plesk Add a Database form with details filled"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Create the Database">
    Click on the **Create Database** button to finalize the creation process.
  </Step>
</Steps>

Upon successful creation, a confirmation message will be displayed, indicating that your database has been successfully created.
