---
title: "Create a New Database User in Plesk"
description: "Learn how to create a new database user in your Plesk control panel."
---

This guide will walk you through the process of creating a new database user in Plesk.

<Steps>
  <Step title="Log in to Plesk">
    Begin by logging into your Plesk account.
  </Step>
  <Step title="Navigate to Databases">
    From the **Sidebar Menu**, click on **Databases**.
    <Frame>
      <img 
        src="/images/kb/plesk-database-menu.gif" 
        alt="Plesk Databases Menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access User Management">
    Click on the **User Management** tab.
    <Frame>
      <img 
        src="/images/kb/plesk-database-user-management-menu.gif" 
        alt="Plesk Database User Management Menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Add New Database User">
    Under the **Database Users** section, click on **Add Database User**.
    <Frame>
      <img 
        src="/images/kb/plesk-add-database-user-button.gif" 
        alt="Plesk Add Database User Button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Fill in User Details">
    In the **Adding Database User** section, provide the following information:
    <ul>
      <li><strong>Database user name:</strong> Enter a desired username (e.g., <code>blog</code>).</li>
      <li><strong>Password</strong>: Enter a password manually or click <strong>Generate</strong>. If you generate a password, click <strong>Show</strong> and copy it to a secure location.</li>
      <li><strong>Database</strong>: Select the relevant database from the dropdown menu.</li>
      <li><strong>Access Control</strong>: Choose the appropriate access level based on your requirements.</li>
    </ul>
    <Frame>
      <img 
        src="/images/kb/plesk-adding-database-user-option.gif" 
        alt="Plesk Adding Database User Options"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Confirm Creation">
    Finally, click <strong>OK</strong> to confirm and create the user.
  </Step>
</Steps>

Your new database user will now be created and ready for use.
