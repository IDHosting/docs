---
title: "Enable the Mailparse Extension of PHP Using CloudLinux Selector in Plesk"
description: "Learn how to enable the Mailparse PHP extension using the CloudLinux Selector in Plesk."
---

You can use the CloudLinux Selector in Plesk to easily enable the **Mailparse** extension for your PHP installation.

<Steps>
  <Step title="Log in to Plesk">
    Access your Plesk control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to PHP Selector">
    In the right-hand sidebar, click on **PHP Selector**.
    <Frame>
      <img
        src="/images/kb/plesk-cl-php-selector-menu.gif"
        alt="Plesk CloudLinux PHP Selector menu item"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Extensions">
    Click on the **Extensions** menu tab.
    <Frame>
      <img
        src="/images/kb/cl-extensions-tab-button.gif"
        alt="CloudLinux Extensions tab button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      If the <strong>Extensions</strong> page displays an error related to the native PHP version, you must first change it to a non-native version before proceeding.
    </Note>
  </Step>
  <Step title="Enable Mailparse">
    Locate **Mailparse** in the list of extensions and check the box next to it.
    <Frame>
      <img
        src="/images/kb/cl-mailparse-enable.gif"
        alt="Enabling the Mailparse extension in CloudLinux"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    Once enabled, a success message will appear, confirming that the **Mailparse** module has been saved.
  </Step>
</Steps>
