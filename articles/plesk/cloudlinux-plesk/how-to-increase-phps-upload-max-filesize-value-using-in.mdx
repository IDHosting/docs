---
title: "Increase PHP's upload_max_filesize Value Using CloudLinux Selector in Plesk"
description: "Learn how to adjust PHP's upload_max_filesize setting using the CloudLinux PHP Selector within Plesk."
---

You can modify PHP's `upload_max_filesize` setting to accommodate larger file uploads through CloudLinux's PHP Selector in Plesk.

<Steps>
  <Step title="Log in to Plesk">
    Access your Plesk account by logging in with your credentials.
  </Step>
  <Step title="Navigate to PHP Selector">
    In the right sidebar, click on the **Pull icon** and select **PHP Selector**.

    <Frame>
      <img 
        src="/images/kb/plesk-cl-php-selector-menu.gif" 
        alt="Plesk CloudLinux PHP Selector menu item"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Options">
    Click on the **Options** menu.

    <Frame>
      <img 
        src="/images/kb/cl-options-menu.gif" 
        alt="CloudLinux Options menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>

    <Note>
      If the **Options** page displays an error related to the native PHP version, you'll need to switch to a non-native PHP version first before proceeding.
    </Note>
  </Step>
  <Step title="Adjust upload_max_filesize">
    Locate the `upload_max_filesize` setting. Click on the drop-down menu next to it and select the desired limit that meets your requirements.

    <Frame>
      <img 
        src="/images/kb/cl-upload-max_filesize.gif" 
        alt="CloudLinux upload_max_filesize setting adjustment"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>

    A confirmation message will appear indicating that the maximum upload file size has been successfully changed.
  </Step>
</Steps>
