---
title: "Enable or Disable PHP Extensions Using the CloudLinux Selector in Plesk"
description: "Learn how to manage PHP extensions in Plesk using the CloudLinux Selector."
---

You can use the CloudLinux Selector to easily enable or disable available PHP extensions for your website within Plesk.

<Steps>
  <Step title="Log in to Plesk">
    Access your Plesk control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to PHP Selector">
    In the right sidebar, click the **Pull icon** and then select **PHP Selector**.
    <Frame>
      <img
        src="/images/kb/plesk-cl-php-selector-menu.gif"
        alt="Plesk sidebar with PHP Selector highlighted"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Extensions">
    Click on the **Extensions** tab.
    <Frame>
      <img
        src="/images/kb/cl-extensions-tab-button.gif"
        alt="PHP Selector interface with Extensions tab highlighted"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      If you encounter an error related to the native PHP version on the Extensions page,
      you may need to switch to a non-native PHP version first.
    </Note>
  </Step>
  <Step title="Enable or Disable Extensions">
    To enable a PHP extension, check the box next to its name. To disable an extension, uncheck the box.
    <Frame>
      <img
        src="/images/kb/cl-enable-disabled-extension.gif"
        alt="Interface for enabling and disabling PHP extensions"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    The interface will indicate whether the module is currently enabled or disabled.
  </Step>
</Steps>
