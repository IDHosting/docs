---
title: "Enable the GD Extension of PHP Using CloudLinux Selector in Plesk"
description: "This guide explains how to enable the GD extension for PHP using the CloudLinux Selector in Plesk."
---

You can use the CloudLinux Selector to enable the GD extension in PHP via Plesk.

<Steps>
  <Step title="Log in to your Plesk account">
    Access your Plesk control panel.
  </Step>
  <Step title="Navigate to PHP Selector">
    Click on the **Pull icon** in the right sidebar and select **PHP Selector**.
    <Frame>
      <img 
        src="/images/kb/plesk-cl-php-selector-menu.gif" 
        alt="Plesk CloudLinux PHP Selector menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Extensions">
    Click on the **Extensions** menu.
    <Frame>
      <img 
        src="/images/kb/cl-extensions-tab-button.gif" 
        alt="CloudLinux Extensions tab button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>If the **Extensions** page displays an error related to the native PHP version, change it to a non-native version first.</Note>
  </Step>
  <Step title="Enable GD Extension">
    Check the box next to **GD**. Once enabled, a success message will appear.
    <Frame>
      <img 
        src="/images/kb/cl-enable-gd-extension.gif" 
        alt="Enable GD extension in CloudLinux"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    The message will confirm that the **GD** module has been saved.
  </Step>
</Steps>
