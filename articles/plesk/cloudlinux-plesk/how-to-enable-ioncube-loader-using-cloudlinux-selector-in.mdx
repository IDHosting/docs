---
title: "Enable ionCube Loader Using CloudLinux Selector in Plesk"
description: "A guide on how to enable the ionCube Loader extension in Plesk using the CloudLinux Selector."
---

You can use the CloudLinux Selector to easily enable the ionCube Loader extension for your Plesk environment. Follow these steps to get it set up.

<Steps>
  <Step title="Log in to Plesk">
    Begin by logging into your Plesk account.
  </Step>
  <Step title="Access PHP Selector">
    In the right sidebar, click on the **Pull icon** and then select **PHP Selector**.
    <Frame>
      <img
        src="/images/kb/plesk-cl-php-selector-menu.gif"
        alt="Plesk CloudLinux PHP Selector menu item"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Navigate to Extensions">
    Click on the **Extensions** menu tab.
    <Frame>
      <img
        src="/images/kb/cl-extensions-tab-button.gif"
        alt="CloudLinux Extensions tab button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Note>
      If the **Extensions** page displays an error related to the native PHP version, you will need to change it to a non-native version first before proceeding.
    </Note>
  </Step>
  <Step title="Enable ionCube Loader">
    Locate and check the box next to **ioncube_loader**. Once selected, a success message will appear indicating that the module has been saved.
    <Frame>
      <img
        src="/images/kb/cl-enable-ioncubeloader.gif"
        alt="Enabling ionCube Loader checkbox"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
    <Frame>
      <img
        src="/images/kb/cl-ioncube-loader-enabled.gif"
        alt="Success message confirming ionCube Loader is enabled"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

You have now successfully enabled the ionCube Loader using the CloudLinux Selector in Plesk.
