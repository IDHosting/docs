## Editing a Scheduled Task

Follow these steps to modify an existing scheduled task in Plesk.

<Steps>
  <Step title="Log in to Plesk">
    Access your Plesk account by logging in with your credentials.
  </Step>
  <Step title="Navigate to Websites & Domains">
    From the **Sidebar Menu**, click on **Websites & Domains**.
    <Frame>
      <img 
        src="/images/kb/plesk-websites-and-domains-menu.gif" 
        alt="Plesk Websites & Domains menu item"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access Scheduled Tasks">
    In the right sidebar, click the **Pull icon** and then select **Scheduled Tasks**.
    <Frame>
      <img 
        src="/images/kb/plesk-scheduled-task-sidebar-menu.gif" 
        alt="Plesk Scheduled Tasks menu option"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Locate and Select Task">
    Under the **Command** list, find the task you wish to edit and click on it.
    <Frame>
      <img 
        src="/images/kb/plesk-task-edit.gif" 
        alt="Plesk task list with a task selected for editing"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Make Changes and Confirm">
    Modify the task settings as needed and click **OK** to save your changes.
    <Frame>
      <img 
        src="/images/kb/plesk-task-edit-option.gif" 
        alt="Plesk task editing options"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

## Deleting a Scheduled Task

To remove a scheduled task from Plesk, follow these steps.

<Steps>
  <Step title="Select Task(s) for Removal">
    In the **Command** list, select the checkbox next to the task(s) you want to delete. Then, click the **Remove** button.
    <Frame>
      <img 
        src="/images/kb/plesk-task-remove.gif" 
        alt="Plesk task list with tasks selected for removal"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Confirm Deletion">
    A confirmation prompt will appear. Click **Yes** to permanently remove the selected task(s).
  </Step>
</Steps>

You have now successfully learned how to edit and delete scheduled tasks in Plesk.
