---
title: "Edit a File in Plesk File Manager"
description: "Learn how to edit website files directly within the Plesk File Manager for quick modifications."
---

<Info>
This guide explains how to edit a file directly in the Plesk File Manager. This is useful for making quick changes to your website files without needing an FTP client.
</Info>

<Steps>
  <Step title="Log in to Plesk">
    Begin by logging in to your Plesk account.
  </Step>
  <Step title="Navigate to Files">
    From the Sidebar Menu, click on **Files**.
    <Frame>
      <img 
        src="/images/kb/plesk-client-files-menu.gif" 
        alt="Plesk client files menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Access httpdocs Directory">
    Under **File Manager**, navigate into the **httpdocs** directory.
    <Frame>
      <img 
        src="/images/kb/plesk-filemanager-httpdocs.gif" 
        alt="Plesk file manager httpdocs directory"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Open File Editor">
    Hover over the file you wish to edit. The corresponding line will highlight. Click the **Menu Icon** <img src="/images/kb/plesk-contextmenu-file-edit.gif" alt="Plesk context menu file edit icon" /> at the end of the line to open the Context Menu. Select **Edit In HTML Editor** or another suitable editor option.
    <Frame>
      <img 
        src="/images/kb/plesk-filemanager-edit-file-contextmenu.gif" 
        alt="Plesk file manager edit file context menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Edit and Save">
    Make your desired changes within the editor. Once finished, click **OK** to confirm and save your modifications.
    <Frame>
      <img 
        src="/images/kb/plesk-editor-ui.gif" 
        alt="Plesk editor UI"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

The file has now been successfully edited and saved within Plesk.
