---
title: "Upload Files via the Plesk File Manager"
description: "A guide on how to upload files to your Plesk server using the File Manager."
---

Follow these steps to upload files using the Plesk File Manager.

<Steps>
  <Step title="Log in to your Plesk account">
    Access your Plesk control panel by logging in with your credentials.
  </Step>
  <Step title="Navigate to Files">
    From the **Sidebar Menu**, click on **Files**.
    <Frame>
      <img
        src="/images/kb/plesk-client-files-menu.gif"
        alt="Plesk client files menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Enter the httpdocs directory">
    Under **File Manager**, navigate into the **httpdocs** directory. This is typically where your website's public files are stored.
    <Frame>
      <img
        src="/images/kb/plesk-filemanager-httpdocs.gif"
        alt="Plesk file manager httpdocs directory"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Upload File">
    Click on the **plus** icon and select **Upload File**. Then, choose the file you wish to upload from your computer.
    <Frame>
      <img
        src="/images/kb/plesk-filemanager-upload-file.gif"
        alt="Plesk file manager upload file option"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Open the file">
    Click **Open** to initiate the file upload.
  </Step>
</Steps>

<Tip>
  You can select multiple files for upload simultaneously. To do this, press and hold the <strong>CTRL</strong> key on your keyboard while clicking on the desired files.
</Tip>
