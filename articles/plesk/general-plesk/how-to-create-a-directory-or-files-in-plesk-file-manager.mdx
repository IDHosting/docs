---

Learn how to create new folders or files in the Plesk File Manager. This is useful for adding and managing files directly from the panel.

## How to Create a Directory in Plesk File Manager

<Steps>
  <Step title="Log in to your Plesk account">
    Access your Plesk control panel.
  </Step>
  <Step title="Navigate to Files">
    Click on **Files** from the Sidebar Menu.
    <Frame>
      <img 
        src="/images/kb/plesk-client-files-menu.gif" 
        alt="Plesk Client Files Menu"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Select Domain (if applicable)">
    If you have multiple domains, click on the appropriate one.
    <Frame>
      <img 
        src="/images/kb/plesk-files-multiple-domains.gif" 
        alt="Plesk Files Multiple Domains"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Choose Directory">
    Under **File Manager**, click on the **httpdocs** directory or any other directory where you want to create the new directory.
    <Frame>
      <img 
        src="/images/kb/plesk-filemanager-httpdocs.gif" 
        alt="Plesk File Manager httpdocs directory"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Initiate Directory Creation">
    Click on the **+plus** icon and choose **Create Directory**.
    <Frame>
      <img 
        src="/images/kb/plesk-client-create-directory.gif" 
        alt="Plesk Create Directory button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Name and Confirm Directory">
    Enter a directory name, such as `my-directory`, in the input box. Then confirm by clicking **OK**.
    <Frame>
      <img 
        src="/images/kb/plesk-directory-name.gif" 
        alt="Plesk Enter Directory Name"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
</Steps>

## How to Create a New File in Plesk File Manager

<Steps>
  <Step title="Initiate File Creation">
    Click on the **+plus** icon and choose **Create File**.
    <Frame>
      <img 
        src="/images/kb/plesk-create-file.gif" 
        alt="Plesk Create File button"
        style={{ width: '100%', height: 'auto' }}
      />
    </Frame>
  </Step>
  <Step title="Name and Confirm File">
    Enter the file name, such as `home.html`, in the input box. Then confirm by clicking **OK**.
  </Step>
</Steps>
